# 3-3. 별의 일생

**학습 목표:** 항성의 탄생부터 죽음까지 진화 과정을 이해한다

## 핵심 개념

- 성운에서 별의 탄생 (중력 수축)
- 주계열성과 핵융합
- 질량에 따른 운명 분기
  - 태양급: 적색거성 → 행성상 성운 → 백색왜성
  - 대질량: 초신성 → 중성자별 또는 블랙홀

## 3D 콘텐츠

- 항성 진화 타임랩스 (질량 슬라이더로 운명 변경)
- HR도(헤르츠스프룽-러셀 도표) 인터랙티브
- 별의 내부 구조 단면도

## XR 강화 요소

- 초신성 폭발 순간 VR 체험
- 블랙홀 근처 시공간 왜곡 시각화
