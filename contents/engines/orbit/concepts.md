# 궤도 엔진 — 핵심 개념

> 소스: 3-1. 지구와 달 + 3-2. 태양계 + 3-3. 별의 일생 (중력 부분) + 3-4. 빛과 스펙트럼 (적색편이) + 3-5. 은하 (중력 구조)

## 개념 목록

| 개념 | 출처 | 엔진 변수와의 연결 |
|------|------|-------------------|
| 자전과 공전 | 3-1 | 회전 속도, 축 기울기 변수 |
| 조석 현상 | 3-1 | 천체 간 거리, 질량 변수 |
| 일식/월식 조건 | 3-1 | 궤도 기울기, 위치 변수 |
| 케플러 법칙 | 3-2 | 타원 궤도 이심률, 주기-거리 관계 |
| 행성 크기/거리 비교 | 3-2 | 스케일 모드 전환 변수 |
| 중력 수축 (별 탄생) | 3-3 | 가스 질량, 밀도 변수 |
| 적색편이 | 3-4 | 후퇴 속도 변수 → 파장 변화 |
| 은하 회전 | 3-5 | 은하 질량, 반경 변수 |

| 변수 | 타입 | 기본값 | 범위 | 설명 |

| `mass` | float[] | [5.97e24, 7.35e22] | — | 천체 질량 (kg) |
| `distance` | float | 384400e3 | — | 천체 간 거리 (m) |
| `velocity` | float[] | — | — | 초기 속도 벡터 (m/s) |

| `eccentricity` | float | 0.0549 | 0 ~ 0.99 | 궤도 이심률 |
| `launchVelocity` | float | — | 0 ~ 20000 | 발사 속도 (m/s, 탈출 시나리오) |
| `galaxyVelocity` | float | 0 | 0 ~ 3e8 | 은하 후퇴 속도 (m/s) |
