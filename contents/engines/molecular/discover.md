# 분자 엔진 — Discover 설명 소재

> 소스: 2-1. 원자와 결합 + 2-2. 분자의 형태 + 2-3. 화학반응 + 2-4. 생체분자
> 개념별 직관적 설명, 비유, 심화 설명의 원재료

## 설명 소재

| 개념 | 직관적 비유 (Level 1) | 개념 설명 (Level 2) |
|------|----------------------|--------------------|
| 공유결합 | "두 사람이 하나의 줄넘기 줄을 함께 잡는 것" | 두 원자가 전자를 공유하여 안정한 상태를 만든다 |
| 이온결합 | "한 사람이 다른 사람에게 선물을 주고, 고마움으로 붙어 있는 것" | 전자를 잃은 양이온과 얻은 음이온이 정전기적으로 끌린다 |
| VSEPR | "풍선 4개를 묶으면 자연스럽게 사면체 모양이 되는 것" | 전자쌍은 서로 최대한 멀어지려 하고, 이것이 분자의 모양을 결정한다 |
| 극성 | "한쪽이 더 힘이 센 줄다리기" | 전기음성도 차이로 전자가 한쪽으로 쏠리면 극성 분자가 된다 |
| 활성화 에너지 | "언덕 너머 내리막길 — 일단 언덕을 넘으면 반응이 저절로 진행" | 반응이 시작되려면 최소한의 에너지(활성화 에너지)가 필요하다 |
| 촉매 | "언덕에 터널을 뚫어 지름길을 만드는 것" | 촉매는 활성화 에너지를 낮춰 반응 속도를 빠르게 한다 |
| 르샤틀리에 | "균형 잡힌 시소에서 한쪽에 무게를 더하면 반대쪽이 올라가는 것" | 평형 상태에 외부 변화를 주면, 그 변화를 상쇄하는 방향으로 이동 |
| DNA 이중나선 | "나선형 사다리 — 가로대가 염기쌍" | A-T, G-C 상보적 결합으로 유전정보를 안정적으로 저장 |

## 개념 연결 (크로스 참조)

- **에너지 양자화** ← 양자 엔진: "원자의 전자가 불연속 에너지 준위에 있는 이유"
- **확률구름/오비탈** ← 양자 엔진: "오비탈의 모양이 화학결합의 방향을 결정"
