# 파동 엔진 — 핵심 개념

> 소스: 1-3. 파동 (핵심) + 1-4. 소리와 빛 (소리 부분) + 3-4. 빛과 스펙트럼 (스펙트럼 부분)

## 개념 목록

| 개념 | 출처 | 엔진 변수와의 연결 |
|------|------|-------------------|
| 횡파와 종파 | 1-3 | 파동 유형 변수 (transverse/longitudinal) |
| 진폭, 주기, 파장, 진동수 | 1-3 | 핵심 변수 4개 |
| 보강/상쇄 간섭 | 1-3 | 파원 수, 위상차 변수 |
| 회절 | 1-3 | 슬릿 폭 vs 파장 비율 |
| 공명 | 1-3 | 진동수 매칭 변수 |
| 음파: 높낮이, 세기, 음색 | 1-4 | 진동수=피치, 진폭=볼륨, 배음=음색 |
| 도플러 효과 | 1-4 | 음원 속도 변수 |
| 전자기 스펙트럼 | 1-4 + 3-4 | 파장 범위 변수 (라디오 ~ 감마선) |
| 흑체복사와 별의 색-온도 | 3-4 | 온도 변수 → 스펙트럼 피크 이동 |
| 적색편이 | 3-4 | 🔗 궤도 엔진 크로스 참조 |

## 엔진 변수

| 변수 | 타입 | 기본값 | 범위 | 설명 |
|------|------|--------|------|------|
| `waveType` | enum | transverse | transverse / longitudinal | 파동 유형 |
| `frequency` | float | 440 | 1 ~ 10000 | 진동수 (Hz) |
| `amplitude` | float | 1.0 | 0.1 ~ 10 | 진폭 |
| `wavelength` | float | — | auto | 파장 (v/f로 자동 계산) |
| `sources` | int | 1 | 1 ~ 5 | 파원 수 |
| `phaseDiff` | float | 0 | 0 ~ 2π | 파원 간 위상차 |
| `slitWidth` | float | 1.0 | 0.1 ~ 10 | 슬릿 폭 (λ 단위) |
| `sourceSpeed` | float | 0 | -343 ~ 343 | 음원 이동 속도 (m/s) |
| `starTemp` | float | 5778 | 2000 ~ 30000 | 별 표면 온도 (K) |
