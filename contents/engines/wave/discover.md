# 파동 엔진 — Discover 설명 소재

> 소스: 1-3. 파동 + 1-4. 소리와 빛 + 3-4. 빛과 스펙트럼
> 개념별 직관적 설명, 비유, 심화 설명의 원재료

## 설명 소재

| 개념 | 직관적 비유 (Level 1) | 개념 설명 (Level 2) |
|------|----------------------|--------------------|
| 간섭 | "물에 돌 두 개를 던지면 파문이 만나는 곳에서 물이 솟거나 잔잔해진다" | 두 파동이 만나면 진폭이 더해짐(보강) 또는 상쇄됨 |
| 도플러 | "구급차가 다가올 때 소리가 높고, 지나가면 낮아진다" | 파원이 움직이면 앞쪽은 파장 축소, 뒤쪽은 파장 확대 |
| 공명 | "그네를 타이밍에 맞춰 밀면 점점 높이 올라간다" | 외부 진동수가 고유 진동수와 일치하면 진폭 폭발 |
| 회절 | "문 뒤에서도 소리가 들리는 것" | 파동이 장애물을 돌아가는 현상. 파장 ≈ 슬릿 폭일 때 가장 뚜렷 |
| 흑체복사 | "달궈진 쇠가 빨갛다가 하얗게 변하는 것" | 온도가 높을수록 방출하는 빛의 피크 파장이 짧아진다 (파란쪽) |

## 개념 연결 (크로스 참조)

- **적색편이** → 궤도 엔진: "파동에서 배운 도플러가 우주 팽창의 증거가 된다"
- **전자기 스펙트럼** → 양자 엔진: "파동의 에너지가 양자화되어 있다"
- **파동 간섭** → 양자 엔진: "물결 간섭과 같은 원리가 양자 세계에서도 작동"
