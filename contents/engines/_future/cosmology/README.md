# 미래 엔진: 항성/우주론

> 소스: 3-3. 별의 일생 (핵융합/진화 부분) + 3-5. 은하 (거대구조) + 3-6. 우주론
> 보관 이유: 궤도 엔진(중력 기반)으로 커버 안 되는 영역. 항성 진화는 시간 스케일, 우주론은 공간 스케일 시뮬레이션이 필요하며, 단순 변수 조합보다 타임랩스/줌아웃 시각화 중심.

## 보관할 핵심 콘텐츠

### 개념

- 별의 탄생 (중력 수축) — 중력 부분은 궤도 엔진에 포함
- 주계열성과 핵융합
- 질량에 따른 운명 분기 (백색왜성/중성자별/블랙홀)
- 우주 거대구조 (필라멘트, 보이드)
- 빅뱅, CMB, 우주 팽창
- 암흑물질/암흑에너지

### 챌린지 아이디어

| 시나리오 | 변수 | 예측 유형 |
|----------|------|-----------|
| 별의 질량 → 최종 운명 | 별 질량 → 백색왜성/중성자별/블랙홀 | pattern |
| HR도 위의 위치 | 별의 특성 → 종류 | pattern |
| 우주 팽창률 → 은하 후퇴 속도 | 허블 상수 → 후퇴 속도 | pattern (🔗 궤도 엔진 크로스) |
