# 미래 엔진: 전자기

> 소스: 1-5. 전기와 자기 + 1-6. 전자기파
> 보관 이유: Phase 2 세로 확장 시 역학 실험실에 "전자기 스테이션"으로 추가 예정. 신규 전자기 엔진이 필요하다.

## 보관할 핵심 콘텐츠

### 개념

- 전하와 전기력 (쿨롱 법칙)
- 전기장 3D 시각화
- 전류, 전압, 저항 (옴의 법칙)
- 직렬/병렬 회로
- 자기장, 전자기 유도 (패러데이 법칙)
- 전자기파 생성/전파
- 전자기 스펙트럼

### 챌린지 아이디어

| 시나리오 | 변수 | 예측 유형 |
|----------|------|-----------|
| 전기장 벡터 시각화 | 전하 배치 → 장 변화 | pattern |
| 회로 조립 시뮬레이터 | 부품 조합 → 전구 밝기 | pattern |
| 전자기 유도 | 자석 이동 속도 → 전류 크기 | pattern |
| 안테나 방사 패턴 | 안테나 형태 → 방사 패턴 | pattern |

### 예상 엔진 변수

- 전하량, 전하 배치, 전압, 저항, 자기장 세기, 코일 감은 수
