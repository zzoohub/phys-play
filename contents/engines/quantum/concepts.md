# 양자 엔진 — 핵심 개념

> 소스: 4-1~4-6 전체 (Track 4 완전 매핑)

## 개념 목록

| 개념 | 출처 | 엔진 변수와의 연결 |
|------|------|-------------------|
| 에너지 양자화 | 4-1 | 에너지 레벨 변수 (불연속) |
| 광전효과 | 4-1 | 빛 진동수, 세기 변수 |
| 이중슬릿 실험 | 4-2 | 슬릿 간격, 입자 종류, 관측 여부 변수 |
| 드브로이 파장 | 4-2 | 질량 변수 → 파장 변화 |
| 파동함수, 확률밀도 | 4-3 | 양자수(n,l,m) 변수 → 오비탈 모양 |
| 중첩 상태 | 4-3 | 관측 전/후 상태 변수 |
| 양자 터널링 | 4-4 | 장벽 높이, 장벽 폭, 입자 에너지 변수 |
| 스핀 | 4-5 | 측정 축 변수 |
| 양자 얽힘 | 4-5 | 측정 축 조합, 통계 변수 |
| 블로흐 구 / 양자 게이트 | 4-6 | 게이트 종류 변수 (H, X, Z, CNOT) |
| 양자 회로 | 4-6 | 게이트 배치 → 결과 확률 |

## 엔진 변수

| 변수 | 타입 | 기본값 | 범위 | 설명 |
|------|------|--------|------|------|
| `slitCount` | int | 2 | 1 ~ 5 | 슬릿 수 |
| `particleType` | enum | photon | photon / electron / neutron | 입자 종류 |
| `observe` | bool | false | true / false | 관측 여부 |
| `barrierHeight` | float | 1.0 | 0.1 ~ 10 | 터널링 장벽 높이 (eV) |
| `barrierWidth` | float | 1.0 | 0.1 ~ 10 | 터널링 장벽 폭 (nm) |
| `particleEnergy` | float | 0.5 | 0.1 ~ 10 | 입자 에너지 (eV) |
| `quantumNumbers` | object | {n:1, l:0, m:0} | — | 양자수 (n, l, m) |
| `measureAxis` | enum[] | [Z] | X / Y / Z | 측정 축 (얽힘용) |
| `gates` | string[] | [] | H / X / Z / CNOT | 양자 게이트 시퀀스 |
| `qubits` | int | 1 | 1 ~ 4 | 큐빗 수 |
