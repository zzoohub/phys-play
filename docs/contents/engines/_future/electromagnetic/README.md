# 미래 엔진: 전자기 v2

> 소스: 1-5. 전기와 자기 + 1-6. 전자기파
> 보관 이유: Phase 2 세로 확장 시 역학 실험실에 "전자기 스테이션"으로 추가 예정
> v2 원칙: Must 3D/XR · 게이미피케이션 핵심 · WOW Point 필수

## v2 설계 철학

전자기장은 본질적으로 3D 벡터장. 전기장선, 자기력선, 전자기파의 E/B 직교 구조는 2D에서 왜곡 없이 표현 불가. **"보이지 않는 힘"을 3D로 시각화하고 손으로 조작하는 것**이 핵심.

---

## 개념

- 전하와 전기력 (쿨롱 법칙) — 3D 전기장선이 전하 배치에 따라 변형
- 전기장 3D 시각화 — 벡터장을 화살표/유선으로 표시
- 전류, 전압, 저항 (옴의 법칙) — 3D 회로 빌더
- 직렬/병렬 회로 — 3D 공간에서 와이어 연결
- 자기장, 전자기 유도 (패러데이 법칙) — 자석 이동 → 3D 자기장 변화 → 전류
- 전자기파 생성/전파 — E/B 직교 진행파의 3D 구조
- 전자기 스펙트럼 — 파장 스케일의 3D 비교

## 챌린지 아이디어

### #1 전기장 조각가 (Electric Field Sculptor)

| 항목 | 내용 |
|------|------|
| **컨셉** | 양전하/음전하를 3D 공간에 자유롭게 배치하여 목표 전기장 패턴을 만든다 |
| **Must 3D** | 전기장은 3D 벡터장. 다이폴의 등전위면은 3D 곡면 |
| **God Hand** | 전하를 손으로 잡아 배치. 전기장선이 실시간으로 변형 |
| **WOW Point** | 전하 배치만으로 — 등전위면이 기이한 3D 형태를 만든다. "보이지 않는 힘의 조각" |
| **XR 증폭** | 전기장 안에 서서 "힘의 방향"을 느낀다. 전하 근처에서 힘 촉각 피드백 |

### #2 번개 경로 (Lightning Path)

| 항목 | 내용 |
|------|------|
| **컨셉** | 구름과 지면 사이에 3D 건물/나무를 배치하여 번개가 어디로 떨어질지 예측 |
| **Must 3D** | 번개는 3D 공간에서 최소 저항 경로를 따른다. 건물 높이와 3D 배치가 결정적 |
| **God Hand** | 건물/피뢰침을 3D로 배치. 전위차를 조절하여 번개를 유도 |
| **WOW Point** | 번개는 직선이 아니다 — 3D 공간에서 지그재그로 갈라지는 스텝 리더(step leader) |
| **XR 증폭** | 번개가 머리 위를 지나가는 것을 슬로모션으로 관찰. 공기가 이온화되는 순간 |

### #3 자기장 놀이터 (Magnetic Field Playground)

| 항목 | 내용 |
|------|------|
| **컨셉** | 자석/전류 루프를 3D 공간에 배치하여 자기장을 시각화하고 물체에 미치는 힘을 관찰 |
| **Must 3D** | 자기력선은 본질적으로 3D 폐곡선. 오른손 법칙은 3D 벡터곱 |
| **God Hand** | 자석을 손으로 이동/회전. 철가루가 3D로 정렬되어 자기장선을 보여준다 |
| **WOW Point** | 두 자석을 특정 각도로 배치하면 — 자기장이 "터널"을 형성하여 입자를 가두는 자기 거울 |
| **XR 증폭** | 자기장 안에서 나침반을 들고 걸어다니며 방향 변화를 체감 |

### #4 전자기 유도 발전소 (EM Induction Power)

| 항목 | 내용 |
|------|------|
| **컨셉** | 코일과 자석을 배치하고 자석을 움직여 전기를 발생시킨다. 발전기의 원리 |
| **Must 3D** | 코일의 면적 벡터와 자기장의 3D 각도가 유도 기전력을 결정. 렌츠의 법칙은 3D 방향 |
| **God Hand** | 자석을 코일 안팎으로 직접 움직인다. 속도에 따라 전구 밝기가 변한다 |
| **WOW Point** | 코일을 90도 회전시키면 전기가 0이 된다! — 자속(flux) 변화가 핵심 |
| **XR 증폭** | 자기력선이 코일을 "관통"하는 것을 시각화. 자석 이동 시 유도 전류의 3D 흐름 |

### #5 안테나 빔 설계 (Antenna Beam Design)

| 항목 | 내용 |
|------|------|
| **컨셉** | 안테나 형태와 배열을 설계하여 특정 방향으로 전자기파를 집중시킨다 |
| **Must 3D** | 방사 패턴은 3D 빔 형태. 다이폴/배열 안테나의 3D 간섭 패턴 |
| **God Hand** | 안테나 요소를 3D로 배치하고 간격/위상을 조절. 방사 패턴이 실시간 변형 |
| **WOW Point** | 여러 안테나를 적절히 배치하면 — 특정 방향으로만 강한 빔! "보이지 않는 에너지의 집중" |
| **XR 증폭** | 안테나 주위를 걸어다니며 전자기파 세기를 체감. 빔 방향에 서면 "강한 신호" |

### #6 전자기파 서핑 (EM Wave Surfing)

| 항목 | 내용 |
|------|------|
| **컨셉** | 전자기파를 타고 이동하며 E/B 벡터의 직교 구조와 진행 방향의 관계를 체험 |
| **Must 3D** | 전자기파는 E(전기장)/B(자기장)/k(진행방향)가 서로 직교하는 3D 구조 |
| **God Hand** | 파원의 진동 방향을 바꾸면 편광이 변한다. 편광 필터를 손으로 회전 |
| **WOW Point** | "빛은 E와 B가 서로를 낳으며 달리는 파동" — 맥스웰의 발견을 직접 본다 |
| **XR 증폭** | 전자기파 위에 서서 E와 B가 번갈아 진동하며 진행하는 것을 라이딩 |

## 예상 엔진 변수

| 변수 | 타입 | 설명 |
|------|------|------|
| `charges` | array | 전하 배치 [{charge, position}] |
| `magnets` | array | 자석 배치 [{type, position, orientation}] |
| `coils` | array | 코일 [{turns, radius, position, axis}] |
| `voltage` | float | 전압 (V) |
| `resistance` | float | 저항 (Ω) |
| `fieldStrength` | float | 자기장 세기 (T) |
| `frequency` | float | 전자기파 진동수 (Hz) |
| `antennaElements` | array | 안테나 요소 [{type, position, phase}] |
| `polarization` | vec3 | 편광 방향 |
| `showFieldLines` | bool | 장선 표시 여부 |
| `showEquipotential` | bool | 등전위면 표시 여부 |
