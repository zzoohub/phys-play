# 미래 엔진: 생체세포 v2

> 소스: 2-4. 생체분자 (기능 부분) + 2-5. 세포 내부 + 2-6. 유전정보
> 보관 이유: 물리 시뮬레이션 엔진 모델에 직접 맞지 않는다. "바이오 탐험 엔진"으로 별도 아키텍처 필요.
> v2 원칙: Must 3D/XR · 게이미피케이션 핵심 · WOW Point 필수

## v2 설계 철학

세포는 **나노스케일의 3D 공장**. 수백 가지 분자 기계가 동시에 작동하는 3D 공간을 "축소되어 걸어다니며" 탐험하는 것이 핵심. 2D 교과서 그림과는 완전히 다른 세계.

---

## 개념

- 세포막 수송 (능동/수동) — 3D 지질 이중층을 관통하는 분자 이동
- 미토콘드리아와 ATP 생산 — 크리스태 내부의 3D 전자전달계
- DNA 복제 — 헬리케이스가 나선을 풀고 DNA 폴리머레이스가 따라가는 3D 과정
- 전사와 번역 — 리보솜이 mRNA를 따라가며 단백질을 조립하는 3D 과정
- 코돈-아미노산 대응 — tRNA가 3D 공간에서 맞는 코돈을 찾아 결합

## 챌린지 아이디어

### #1 세포막 게이트키퍼 (Membrane Gatekeeper)

| 항목 | 내용 |
|------|------|
| **컨셉** | 지질 이중층 문지기가 되어 분자 크기/전하/극성에 따라 통과 여부를 결정 |
| **Must 3D** | 지질 이중층은 3D 구조. 채널 단백질의 3D 구멍 크기가 선택성을 결정 |
| **God Hand** | 분자를 손으로 잡아 막에 던져본다. 채널 단백질을 열고 닫는다 |
| **WOW Point** | 물 분자는 비극성 막을 못 통과하지만, 아쿠아포린 채널이 있으면 초고속 통과! |
| **XR 증폭** | 지질 이중층 사이에 서서 인지질 꼬리가 출렁이는 것을 체감 |

### #2 리보솜 조립 공장 (Ribosome Assembly Line)

| 항목 | 내용 |
|------|------|
| **컨셉** | mRNA 코돈을 읽고 올바른 tRNA를 가져와 단백질 사슬을 조립 |
| **Must 3D** | 리보솜의 대소 서브유닛, tRNA의 3D 구조, 펩티드 결합의 공간적 형성 |
| **God Hand** | tRNA를 손으로 잡아 리보솜의 A 사이트에 끼워넣는다. 안티코돈이 맞으면 결합 |
| **WOW Point** | 실수로 틀린 tRNA를 넣으면 — 돌연변이 단백질! 아미노산 하나 차이가 낫모양 적혈구병 |
| **XR 증폭** | 리보솜 내부에 서서 단백질 사슬이 한 아미노산씩 길어지는 과정을 관찰 |

### #3 미토콘드리아 전력 발전소 (Mitochondria Power Plant)

| 항목 | 내용 |
|------|------|
| **컨셉** | 전자전달계를 따라 전자가 이동하며 H+ 그래디언트를 만들고 ATP를 합성하는 과정 |
| **Must 3D** | 크리스태의 접힌 3D 구조가 표면적을 극대화. ATP 합성효소의 3D 회전 모터 |
| **God Hand** | 전자를 전달체 사이로 이동시킨다. ATP 합성효소가 H+ 흐름으로 회전하며 ATP 조립 |
| **WOW Point** | ATP 합성효소는 분자 크기의 회전 모터! 초당 수백 번 회전하며 ATP를 찍어낸다 |
| **XR 증폭** | 크리스태 막 위에 서서 ATP 합성효소의 회전을 관찰. 양성자 폭포를 체감 |

### #4 DNA 복제 레이스 (DNA Replication Race)

| 항목 | 내용 |
|------|------|
| **컨셉** | 헬리케이스/프라이메이스/폴리머레이스 팀을 조율하여 DNA를 복제 |
| **Must 3D** | 복제 포크의 3D 구조. 선도 가닥과 지연 가닥의 비대칭이 3D에서만 이해 가능 |
| **God Hand** | 각 효소를 올바른 위치에 배치하고 가동. 오카자키 단편 연결 |
| **WOW Point** | 선도 가닥은 연속, 지연 가닥은 거꾸로 조각조각! — 3D에서 보면 "왜"가 직관적으로 이해됨 |
| **XR 증폭** | 복제 포크 안에 서서 나선이 풀리고 새 가닥이 합성되는 과정을 목격 |

## 예상 엔진 변수

| 변수 | 타입 | 설명 |
|------|------|------|
| `cellType` | enum | 세포 종류 (prokaryote/eukaryote) |
| `organelle` | enum | 소기관 (membrane/mitochondria/ribosome/nucleus) |
| `molecules` | array | 분자 목록 [{type, size, charge, polarity}] |
| `mRNA_sequence` | string | mRNA 코돈 서열 |
| `channels` | array | 채널 단백질 [{type, selectivity}] |
| `dnaSequence` | string | DNA 서열 |
| `enzymes` | array | 효소 배치 [{type, position}] |
| `showGradient` | bool | H+ 그래디언트 시각화 |
| `showATPcount` | bool | ATP 생성 카운터 표시 |
| `timeScale` | float | 시뮬레이션 속도 배율 |
