# 미래 엔진: 생체세포

> 소스: 2-4. 생체분자 (기능 부분) + 2-5. 세포 내부 + 2-6. 유전정보
> 보관 이유: 물리 시뮬레이션 엔진 모델에 직접 맞지 않는다. "변수 조합으로 챌린지 생성" 패턴보다 시각적 탐험/애니메이션 중심. 별도의 "바이오 탐험 엔진"이 필요할 수 있으며, 엔진 아키텍처 확장이 필요하다.

## 보관할 핵심 콘텐츠

### 개념

- 단백질 접힘 (1차→4차 구조) — 구조 부분은 분자 엔진에 포함
- 세포막 수송 (능동/수동)
- 미토콘드리아와 ATP 생산
- DNA 복제, 전사, 번역
- 코돈-아미노산 대응

### 챌린지 아이디어 (엔진 모델 적용 가능한 것)

| 시나리오 | 변수 | 예측 유형 |
|----------|------|-----------|
| 세포막 수송 | 분자 크기/농도 → 통과 여부 | binary |
| 코돈 번역 | mRNA 서열 → 아미노산 서열 | pattern |
| 돌연변이 영향 | 염기 변경 → 단백질 변화 | pattern |

### 참고

- 2-4의 PDB 기반 단백질 3D 구조 탐험은 분자 엔진의 확장으로 처리 가능
