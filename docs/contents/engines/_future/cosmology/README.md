# 미래 엔진: 항성/우주론 v2

> 소스: 3-3. 별의 일생 (핵융합/진화 부분) + 3-5. 은하 (거대구조) + 3-6. 우주론
> 보관 이유: 궤도 엔진(중력 기반)으로 커버 안 되는 영역. 시간/공간 스케일 시뮬레이션 필요.
> v2 원칙: Must 3D/XR · 게이미피케이션 핵심 · WOW Point 필수

## v2 설계 철학

**시간과 공간의 극단**을 3D로 체험. 별의 일생(수십억 년)을 타임랩스로, 우주 거대구조(수억 광년)를 줌아웃으로. "나"라는 스케일에서는 절대 볼 수 없는 것을 3D/XR로 체험.

---

## 개념

- 별의 탄생 (중력 수축) — 3D 가스 구름의 중력 붕괴
- 주계열성과 핵융합 — 별 내부의 3D 구조 (핵/복사층/대류층)
- 질량에 따른 운명 분기 — 3D HR도 위에서 별의 진화 경로
- 초신성 폭발 — 3D 충격파의 팽창
- 백색왜성/중성자별/블랙홀 — 극단적 밀도의 3D 시각화
- 우주 거대구조 (필라멘트, 보이드) — 3D 우주 지도
- 빅뱅과 CMB — 3D 전방위 마이크로파 배경복사
- 우주 팽창과 허블 법칙 — 풍선 위의 점 비유를 3D로
- 암흑물질/암흑에너지 — 보이지 않는 질량의 3D 분포

## 챌린지 아이디어

### #1 별의 일생 시뮬레이터 (Stellar Life Simulator)

| 항목 | 내용 |
|------|------|
| **컨셉** | 별의 초기 질량을 설정하고 수십억 년의 진화를 타임랩스로 관찰. 최종 운명 예측 |
| **Must 3D** | 별 내부의 층상 구조(핵/복사층/대류층), 적색거성 팽창, 행성상 성운의 3D 구조 |
| **God Hand** | 초기 질량 슬라이더 대신 — 가스 구름을 손으로 "모아" 별을 만든다. 많이 모을수록 무거운 별 |
| **WOW Point** | 무거운 별은 빛나는 시간이 짧다! 태양의 10배 질량 → 수명 1/1000. "빛나는 것은 빨리 죽는다" |
| **XR 증폭** | 별 내부로 들어가서 핵융합이 일어나는 핵을 관찰. 타임랩스로 별이 부풀어 적색거성이 되면 자신이 별 안에 갇힘 |

### #2 초신성 폭발 (Supernova Detonation)

| 항목 | 내용 |
|------|------|
| **컨셉** | 무거운 별의 마지막 순간 — 철 핵 붕괴 → 반동 충격파 → 초신성 폭발을 목격 |
| **Must 3D** | 충격파의 3D 전파, 비대칭 폭발, 중성자별/블랙홀 잔해 |
| **God Hand** | 시간 다이얼로 폭발 순간을 슬로모션/되감기. 별의 단면을 잘라보기 |
| **WOW Point** | 폭발 순간 우주에서 가장 밝다! 은하 전체보다 밝은 빛 → 그리고 남은 것은 찻숟가락 크기에 산 질량의 물질(중성자별) |
| **XR 증폭** | 안전거리에서 초신성 폭발을 목격. 충격파가 지나가는 순간의 밝기 변화 |

### #3 블랙홀 탐험 (Black Hole Explorer)

| 항목 | 내용 |
|------|------|
| **컨셉** | 블랙홀 주변을 탐사선으로 비행하며 시공간 왜곡, 강착원반, 중력 렌즈를 관찰 |
| **Must 3D** | 강착원반의 3D 구조, 중력 렌즈로 뒤쪽 별빛이 휘는 것, 시공간 그리드의 왜곡 |
| **God Hand** | 탐사선을 조종하여 블랙홀에 접근. 사건의 지평선에 가까울수록 시간이 느려지는 것을 체감 |
| **WOW Point** | 블랙홀 뒤의 별이 보인다! — 빛이 휘어서 앞으로 돌아온다 (중력 렌즈) |
| **XR 증폭** | 블랙홀 근처에서 시공간 격자가 찢어지듯 휘는 것을 관찰. 강착원반의 열기 |

### #4 우주 거대구조 탐험 (Cosmic Web Explorer)

| 항목 | 내용 |
|------|------|
| **컨셉** | 은하 → 은하단 → 초은하단 → 필라멘트 → 관측 가능 우주까지 줌아웃 |
| **Must 3D** | 우주 거대구조(필라멘트+보이드)는 본질적으로 3D 거품(foam) 구조 |
| **God Hand** | 핀치 줌으로 스케일 이동. 각 스케일에서 대표 구조를 선택하여 상세 탐험 |
| **WOW Point** | 가장 큰 구조(헤르쿨레스-코로나 보레알리스 장벽, 100억 광년)에서 줌아웃하면 — 균일하다! "우주 원리" |
| **XR 증폭** | 보이드(빈 공간) 안에 서서 사방이 비어있는 공허함 → 필라멘트로 나오면 은하들이 빽빽 |

### #5 CMB 타임머신 (CMB Time Machine)

| 항목 | 내용 |
|------|------|
| **컨셉** | 우주 마이크로파 배경복사(CMB)의 3D 전방위 맵을 분석하여 초기 우주를 추론 |
| **Must 3D** | CMB는 전천구(全天球)의 3D 온도 맵. 미세 요동(anisotropy)이 3D 분포 |
| **God Hand** | CMB 구면 위에서 온도 차이를 손으로 "만져본다". 요동 패턴에서 물질 밀도 불균일 찾기 |
| **WOW Point** | 38만 년 전 우주의 "사진"! 이 미세한 온도 차이(10만분의 1도)에서 은하가 태어났다 |
| **XR 증폭** | CMB 구면 안에 서서 사방에서 오는 138억 년 전의 빛에 둘러싸인 체험 |

## 예상 엔진 변수

| 변수 | 타입 | 설명 |
|------|------|------|
| `starMass` | float | 별 초기 질량 (태양질량 단위) |
| `starAge` | float | 별 현재 나이 (억 년) |
| `timeScale` | float | 타임랩스 배율 |
| `showInternalStructure` | bool | 별 내부 구조 표시 |
| `blackHoleMass` | float | 블랙홀 질량 (태양질량 단위) |
| `probePosition` | vec3 | 탐사선 3D 위치 |
| `probeVelocity` | vec3 | 탐사선 3D 속도 |
| `showSpacetimeGrid` | bool | 시공간 격자 표시 |
| `showAccretionDisk` | bool | 강착원반 표시 |
| `cosmicScale` | float | 우주 관측 스케일 (Mpc) |
| `showFilaments` | bool | 필라멘트 구조 표시 |
| `cmbResolution` | float | CMB 맵 해상도 |
| `showAnisotropy` | bool | CMB 비등방성 표시 |
| `hubbleConstant` | float | 허블 상수 (km/s/Mpc) |
