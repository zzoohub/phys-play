# 양자 엔진 — Discover 설명 소재 v2

> 소스: 4-1~4-6 전체
> v2: 챌린지 v2에 맞춰 업데이트. 3D/XR 체험 기반 설명 강화

## 설명 소재

| 개념 | 직관적 비유 (Level 1) | 개념 설명 (Level 2) |
|------|----------------------|--------------------|
| 이중슬릿 / 자기 간섭 | "공을 두 문으로 동시에 던질 수 있다면? 양자 세계에서는 가능!" | 입자가 측정 전까지 확률파로 존재하며 두 경로를 '동시에' 지나간다. P = \|ψ₁+ψ₂\|² ≠ \|ψ₁\|²+\|ψ₂\|² |
| 관측자 효과 | "보면 달라진다 — 카메라 앞에서 행동이 달라지는 것과 비슷하지만, 더 근본적" | 관측 = 상호작용. 경로 정보를 얻으면 간섭이 사라진다. 양자역학의 '측정 문제' |
| 양자 터널링 | "벽에 공을 던지면 100번 중 1번은 통과하는 유령 공" | 파동함수가 장벽 안에서 지수적으로 감쇠하지만 0이 되지 않아 반대편 발견 확률 존재. 알파붕괴, STM의 원리 |
| 오비탈 | "전자가 '이 근처에 있을 확률이 높은 구름' — 궤도가 아니라 확률" | 양자수(n,l,m)이 확률구름의 3D 모양을 결정. 노드면에서 전자 발견 확률 = 0 |
| 불확정성 원리 | "위치를 정확히 알수록 속도는 더 모른다 — 우주의 근본적 한계" | ΔxΔp ≥ ℏ/2. 실험 장치의 한계가 아닌 자연의 법칙. 파속(wave packet)이 좁아지면 운동량 성분이 넓어짐 |
| 얽힘 | "양말 상자 비유 — 하지만 양자는 열기 전까지 색이 정해지지 않았다" | 얽힌 입자쌍은 측정 전 상태 미결정. 벨 부등식 위반 → 숨은 변수 이론 불가. 비국소적 상관 |
| 슈뢰딩거의 고양이 | "열기 전에는 살아있으면서 동시에 죽어있다?" | 거시적 양자 중첩의 역설. 결어긋남으로 τ~10⁻²⁶초 안에 거시적 중첩은 깨진다 |
| 양자 지우개 | "지운 정보가 돌아오면 간섭도 돌아온다 — 시간이 뒤바뀐 것 같은 기이함" | 경로 정보 ↔ 간섭의 상보적 관계. 지연 선택 = 측정 순서가 아닌 정보 존재 여부가 핵심 |
| 블로흐 구 / 양자 게이트 | "지구본에서 북극이 0, 남극이 1 — 적도는 중첩" | 큐빗의 모든 상태를 3D 구면의 한 점으로 표현. 게이트 = 구면 위의 회전 |
| 광전효과 | "센 빨간 빛보다 약한 자외선이 전자를 내보낸다 — 세기가 아니라 색깔!" | E=hf. 에너지는 진동수에 비례, 세기가 아닌 진동수가 임계값(일함수)을 넘어야 전자 방출 |

## 개념 연결 (크로스 참조)

- **파동 간섭** ← 파동 엔진 #1 소리의 방: "물결/소리 간섭과 같은 원리가 양자 세계에서도 작동"
- **전자기 스펙트럼** ← 파동 엔진: "파동의 에너지가 양자화되어 있다"
- **확률구름/오비탈** → 분자 엔진 #1 분자 조각가: "오비탈의 모양이 화학결합의 방향을 결정"
- **광전효과** → 분자 엔진: "원자의 에너지 준위가 분자 결합 에너지를 결정"
- **터널링** → 분자 엔진 #4 반응 극장: "양자 터널링이 일부 화학 반응을 가능하게 한다"
