# 양자 엔진 — Discover 설명 소재

> 소스: 4-1~4-6 전체
> 개념별 직관적 설명, 비유, 심화 설명의 원재료

## 설명 소재

| 개념 | 직관적 비유 (Level 1) | 개념 설명 (Level 2) |
|------|----------------------|--------------------|
| 이중슬릿 | "공을 두 문으로 동시에 던질 수 있는 것처럼, 입자는 두 슬릿을 '동시에' 통과한다" | 양자 입자는 측정 전까지 확률파로 존재하며, 두 경로의 확률이 간섭한다 |
| 터널링 | "벽에 공을 던지면 100번 중 1번은 벽을 통과하는 것" | 파동함수가 장벽 안에서 0이 아닌 값을 가져서, 반대편에서 발견될 확률이 존재 |
| 얽힘 | "빨간 양말과 파란 양말을 상자에 넣고 보냈을 때, 한 상자를 열면 다른 상자 색을 아는 것. 단, 양자는 열기 전까지 색이 정해지지 않는다" | 얽힌 입자쌍은 측정 전까지 상태가 미결정이며, 한쪽 측정이 다른 쪽의 상태를 즉시 결정한다 |
| 블로흐 구 | "지구본에서 북극이 0, 남극이 1, 적도가 중첩 상태" | 큐빗의 모든 가능한 상태를 구면 위의 한 점으로 표현 |
| 광전효과 | "아무리 센 빨간 빛을 비춰도 전자가 안 나오는데, 약한 자외선으로는 전자가 나온다" | 빛의 에너지는 진동수에 비례하며, 세기가 아닌 진동수가 임계값을 넘어야 한다 |
| 오비탈 | "전자가 '이 근처에 있을 확률이 높은 구름'" | 파동함수의 제곱이 확률밀도이며, 양자수에 따라 구름의 모양이 달라진다 |

## 개념 연결 (크로스 참조)

- **파동 간섭** ← 파동 엔진: "물결 간섭과 같은 원리가 양자 세계에서도 작동"
- **전자기 스펙트럼** ← 파동 엔진: "파동의 에너지가 양자화되어 있다"
- **확률구름/오비탈** → 분자 엔진: "오비탈의 모양이 화학결합의 방향을 결정"
