# 충돌/에너지 엔진 — 핵심 개념 v2

> 소스: 1-2. 에너지와 일 + 3D/XR 필수 확장

## 개념 목록

| 개념 | 챌린지 연결 | 3D/XR 관점 |
|------|-----------|-----------|
| 위치에너지와 운동에너지 | #1 마블런, #2 철거 공 | 3D 트랙/구조물에서 에너지 전환을 실시간 시각화 |
| 에너지 보존 법칙 | #1 마블런, #7 루브 골드버그 | 3D 체인을 따라 에너지 형태가 변환되는 것을 추적 |
| 마찰에 의한 에너지 손실 | #1 마블런, #7 루브 골드버그 | 열 에너지 시각화(3D 파티클)로 "사라진" 에너지를 보이게 |
| 뉴턴 제3법칙 (작용-반작용) | #6 트럭 vs 경차 | 양쪽 시점 3D 전환으로 반직관적 동등성 체험 |
| 3D 운동량 보존 | #3 우주 당구 | 3축 운동량 벡터 보존. 2D에서 불가능한 3D 산란 |
| 탄성/비탄성 충돌 | #3 우주 당구, #6 트럭 vs 경차 | 3D 충돌 변형과 에너지 손실 시각화 |
| 단순 조화 운동과 주기 | #4 진자 웨이브, #5 푸코 진자 | 진자 길이→주기 관계가 매크로 3D 패턴을 만듦 |
| 코리올리 힘과 지구 자전 | #5 푸코 진자 | 지구의 3D 자전과 진자면의 기하학 |
| 충격량과 힘의 관계 | #8 달걀 낙하 | 3D 보호 구조물 설계로 충돌 시간 연장 |
| 에너지 변환 체인 | #7 루브 골드버그 | PE→KE→회전KE→탄성PE→열 의 3D 체인 |
| 비선형 관계 (에너지∝v²) | #2 철거 공 | 높이 2배 ≠ 파괴력 2배. 시각적 극적 차이 |

## 엔진 변수

| 변수 | 타입 | 기본값 | 범위 | 설명 |
|------|------|--------|------|------|
| `height` | float | 10 | 1~50 | 시작 높이 (m) |
| `friction` | float | 0 | 0~1 | 마찰 계수 |
| `loopRadius` | float | 3 | 1~10 | 루프 반지름 (m) |
| `mass` | float[] | [1, 1] | 0.1~100 | 충돌 물체 질량 (kg) |
| `velocity` | vec3[] | [[5,0,0], [0,0,0]] | -50~50 | 충돌 물체 초기 속도 (m/s) — **3D 확장** |
| `elasticity` | float | 1.0 | 0~1 | 탄성 계수 |
| `pendulumLength` | float | 2 | 0.5~10 | 진자 길이 (m) |
| `pendulumAngle` | float | 45 | 5~90 | 진자 초기 각도 (도) |
| `trackPieces` | array | [] | - | 마블런 트랙 조각 — **v2 추가** |
| `structure` | object | null | - | 3D 구조물 정의 — **v2 추가** |
| `balls3D` | array | [] | - | 3D 공 배치 [{mass, position, velocity}] — **v2 추가** |
| `pendulums` | array | [] | - | 다중 진자 [{length, position}] — **v2 추가** |
| `latitude` | float | 48.9 | -90~90 | 푸코 진자 위치 (도) — **v2 추가** |
| `vehicles` | array | [] | - | 충돌 차량 [{mass, velocity, shape}] — **v2 추가** |
| `chainElements` | array | [] | - | 체인 리액션 요소 — **v2 추가** |
| `protectionMaterials` | array | [] | - | 달걀 보호 재료 — **v2 추가** |
| `dropHeight` | float | 1 | 1~50 | 낙하 높이 (m) — **v2 추가** |
