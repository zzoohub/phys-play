# 충돌/에너지 엔진 — 챌린지 시나리오 v2

> **설계 원칙:** 에너지는 "보이지 않는 것"이라 3D에서 시각화할 때 가장 강력하다.
> "만들고 부수는" 인터랙션이 에너지 전환의 직관을 만든다.
> 단순 Yes/No 반복이 아닌, 구축→예측→관찰의 창작적 루프.

## 시나리오 풀 (8개)

### #2 — 철거 공 (Wrecking Ball)

| 항목 | 내용 |
|------|------|
| **핵심 세팅** | 체인에 매달린 철거 공(mass=500kg, 체인 길이=10m). 앞에 3D 블록 구조물(벽, 타워, 아치 등 다양한 구성). 중력=9.8 |
| **예측 유형** | pattern — "이 높이에서 놓으면 구조물이 어느 정도 파괴될까?" 파괴 수준 3단계 선택 (약간 손상 / 부분 파괴 / 완전 붕괴) |
| **God Hand** | 철거 공을 잡아 원하는 높이와 방향으로 당김(체인이 팽팽해지는 피드백) → 놓기. XR: 공을 두 손으로 밀어 올리고 놓기 |
| **WOW 포인트** | 높이를 2배로 올리면 파괴력이 2배가 아니라 **속도의 제곱**에 비례! h=5m vs h=10m의 차이가 극적. 같은 높이에서 충돌 각도를 바꾸면 파괴 패턴이 완전히 다름. 구조물의 약점(키스톤, 지지대)을 맞추면 전체 붕괴 — 구조 공학의 직관 |
| **3D 필수 이유** | 3D 구조물의 물리 기반 파괴. 철거 공의 접근 각도(정면/측면/위)에 따라 파괴 패턴이 3D로 달라짐. 구조물 뒤편의 파편 산포도 3D |
| **XR 증폭** | 거대한 철거 공을 밀어 올리는 무게감. 구조물이 무너지며 파편이 내 쪽으로 날아오는 시각적/청각적 임팩트 |
| **Discover 핵심** | 위치에너지 → 운동에너지 전환 (mgh = ½mv²). 에너지와 속도의 비선형 관계. 구조물의 힘 분산 원리 |
| **difficulty** | 2 |
| **tags** | `potential-energy`, `kinetic-energy`, `energy-transfer`, `destruction`, `nonlinear` |

### #4 — 진자 웨이브 머신 (Pendulum Wave)

| **핵심 세팅** | 길이가 조금씩 다른 15개 진자가 일렬(또는 원형, 3D 나선형)로 배열. 동시에 놓으면 환각적 패턴 생성 |
| **예측 유형** | pattern — "10초/30초/60초 후 진자들이 어떤 패턴을 만들까?" 3개 타임랩스 애니메이션 미리보기 중 선택 |
| **God Hand** | 모든 진자를 한 줄로 잡아 당김(잡는 바) → 동시에 놓기. 놓은 후 시간 배속/감속 조절. XR: 진자 바를 양손으로 잡아 당기고 놓기 |
| **WOW 포인트** | 단순한 진자 15개가 만드는 패턴이 **환각적으로 아름답다**. 웨이브 → 뱀 → 클러스터 → 다시 일렬! 자연에서 복잡한 패턴이 단순한 규칙에서 나온다는 것의 시각적 증명. 이것은 순수한 시각적 경이로움 |
| **3D 필수 이유** | 3D 배열(원형/나선형)에서는 2D에서 볼 수 없는 새로운 패턴이 나타남. 진자를 위에서, 옆에서, 안에서 볼 때 패턴이 다르게 보임 — 관점에 따른 패턴 변화 |
| **XR 증폭** | 진자들 사이에 서서 관찰. 머리 위에서 내려다보면 원형 패턴, 옆에서 보면 웨이브, 비스듬히 보면 또 다른 패턴. 시점에 따른 패턴의 다양성을 신체적으로 체험 |
| **Discover 핵심** | 단순 조화 운동의 주기와 진자 길이의 관계 (T = 2π√(L/g)). 주기의 미세한 차이가 위상차를 만들고, 위상차가 매크로 패턴을 만든다 |

### #5 — 푸코 진자 (Foucault Pendulum)

| 항목 | 내용 |
|------|------|
| **핵심 세팅** | 거대한 돔 건물 천장에 매달린 진자(길이=67m, 질량=28kg). 바닥에 모래 원형 트레이. 지구 자전 시뮬레이션. 위치: 파리(위도 48.9°) |
| **예측 유형** | pattern — "6시간 후 진자의 진동 방향이 어떻게 변해 있을까?" 진동면 회전 각도 3개 중 선택 (0°/45°/90°) |
| **God Hand** | 진자를 잡아 당겨 놓기. 시간 배속 슬라이더로 6시간을 압축 관찰. 위도 슬라이더로 극지/적도 비교. XR: 돔 아래에 서서 진자를 놓고, 시간을 가속하며 바닥의 모래 자국이 회전하는 것을 관찰 |
| **WOW 포인트** | 진자는 같은 방향으로 계속 흔들리는데, 바닥의 모래 자국이 돌아가 있다! "진자가 회전한 게 아니라 지구가 진자 아래에서 돌아간 거다!" 극지에서는 24시간에 360° 회전, 적도에서는 회전 없음. 위도에 따라 다르다! |
| **3D 필수 이유** | 지구 자전축, 진자 진동면, 위도의 기하학적 관계가 본질적으로 3D. 왜 위도에 따라 회전 속도가 다른지는 3D 구 위에서만 직관적으로 보임 |
| **XR 증폭** | 거대한 돔 내부의 스케일감. 진자가 천천히 흔들리는 동안 주변의 모래 자국이 조금씩 어긋나는 것을 발견하는 순간의 소름. 시간 가속 시 지구가 발 아래에서 돌아가는 것을 투명 바닥으로 확인 |
| **Discover 핵심** | 코리올리 힘의 또 다른 표현. 관성 좌표계 vs 회전 좌표계. "1851년 레옹 푸코가 이 실험으로 지구 자전을 증명했다" |
| **difficulty** | 4 |
| **tags** | `foucault`, `earth-rotation`, `coriolis`, `inertial-frame`, `history` |

### #6 — 트럭 vs 경차 (Newton's Third Law Crash)

| 항목 | 내용 |
|------|------|
| **핵심 세팅** | 10톤 트럭(v=60km/h)과 1톤 경차(v=60km/h)의 정면 충돌. 3D 변형 시뮬레이션. 양쪽 차량에 가속도계/힘 게이지 시각화 |
| **예측 유형** | binary — "어느 쪽이 더 큰 힘을 받을까?" (트럭 / 경차 / 같다) |
| **God Hand** | 양쪽 차량의 속도를 드래그로 조절 → "충돌" 버튼. 충돌 후 양쪽 차량 내부/외부 시점 전환. XR: 트럭 운전석 / 경차 운전석에 번갈아 탑승하여 충돌 체험 |
| **WOW 포인트** | 거의 100%의 사람이 "경차가 더 큰 힘을 받는다"고 답한다. **정답: 같은 힘.** 힘 게이지가 양쪽 모두 동일한 값을 표시! "그럼 왜 경차만 찌그러지는 거야?" → F=ma. 같은 F, 10배 작은 m = 10배 큰 가속도(감속). "경차 탑승자가 10배 큰 충격을 느끼는 이유가 이거다!" |
| **3D 필수 이유** | 3D 차량 변형 시뮬레이션. 양쪽 시점을 3D로 전환하며 관찰. 충돌 각도를 바꾸면(정면/측면/사선) 파괴 패턴이 3D로 달라짐 |
| **XR 증폭** | 트럭 운전석에 앉았을 때의 "별로 안 느껴지는" 충돌 vs 경차 운전석에서의 격렬한 충돌을 대비 체험. 뉴턴 제3법칙의 반직관을 신체적으로 |
| **Discover 핵심** | 뉴턴 제3법칙: 작용-반작용은 항상 동일. F=ma에서 같은 F에 대해 m이 작으면 a가 크다. 교통안전과의 연결 |
| **difficulty** | 1 |
| **tags** | `newtons-third-law`, `force`, `mass`, `acceleration`, `counter-intuitive` |

### #7 — 루브 골드버그 체인 (Rube Goldberg Chain Reaction)

| 항목 | 내용 |
|------|------|
| **핵심 세팅** | 에너지 전환 체인 장치: 도미노 → 구슬 낙하 → 시소 → 진자 → 미끄럼틀 → 최종 타깃. 각 구간에서 에너지 형태가 변환됨 (위치→운동→회전→탄성→위치→운동) |
| **예측 유형** | placement — "이 체인이 어디서 멈출까?" 멈추는 지점 선택 + binary — "끝까지 갈까?" |
| **God Hand** | 체인의 마지막 단계를 직접 배치(위치/각도 조절) → 첫 도미노를 밀어 시작. 또는 체인 중간에 새 장치를 추가하여 "끊어진 체인"을 이어가기. XR: 장치 주변을 걸어다니며 관찰, 도미노를 손가락으로 밀기 |
| **WOW 포인트** | 에너지가 위치→운동→회전→탄성→열 등 끊임없이 형태를 바꾸며 전달되는 것이 실시간 에너지 바로 보인다. "에너지는 절대 사라지지 않는다 — 형태만 바뀐다!" 체인이 예상 못한 곳에서 멈추는 순간: "여기서 마찰이 에너지를 잡아먹었구나!" |
| **3D 필수 이유** | 체인이 3D 공간에서 전개 (위층에서 아래층으로, 회전 구조물, 나선 미끄럼틀). 위에서, 옆에서, 안에서 다른 관점으로 에너지 흐름 추적. 2D에서는 복잡한 3D 체인 구조 표현 불가 |
| **XR 증폭** | 체인 옆을 따라 걸으며 에너지 흐름을 실시간 추적. 도미노가 눈앞에서 쓰러지고, 구슬이 머리 위를 지나가고, 시소가 옆에서 기우는 체감 |
| **Discover 핵심** | 에너지 보존의 종합 체험. 에너지 변환 체인: PE → KE → 회전 KE → 탄성 PE → 열에너지. 비가역적 변환(마찰→열)이 왜 "낭비"인지 |
| **difficulty** | 3 |
| **tags** | `energy-conservation`, `energy-transformation`, `chain-reaction`, `friction`, `creative` |

### #8 — 달걀 낙하 보호 (Egg Drop Challenge)

| 항목 | 내용 |
|------|------|
| **핵심 세팅** | 달걀 1개. 보호 재료 팔레트: 스폰지, 종이컵, 풍선, 고무줄, 빨대, 종이. 낙하 높이: 1m → 3m → 5m → 10m 단계별 |
| **예측 유형** | binary — "이 보호 장치로 X미터에서 생존할까?" + pattern — "어떤 설계가 가장 높은 곳에서 살아남을까?" |
| **God Hand** | 재료를 잡아 달걀 주변에 3D로 배치/감싸기. 완성 후 높이를 선택해 떨어뜨리기. XR: 양손으로 재료를 조합하여 달걀을 감싸기, 높이 선택 후 손 놓기 |
| **WOW 포인트** | 같은 높이에서 딱딱한 케이스는 실패, 푹신한 구조는 성공! "왜?" → 충격량(impulse)은 같지만, 충돌 시간을 늘리면 힘이 줄어든다. F = Δp/Δt. "에어백의 원리가 이거다!" 10m에서도 살아남는 설계를 만드는 성취감 |
