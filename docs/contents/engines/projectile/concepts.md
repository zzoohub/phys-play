# 투사체 엔진 — 핵심 개념

> 소스: 1-1. 운동과 힘

## 개념 목록

| 개념 | 출처 | 엔진 변수와의 연결 |
|------|------|-------------------|
| 속도, 가속도, 변위의 관계 | 1-1 핵심 개념 | 발사 속도, 각도 → 궤적 변화 |
| 뉴턴 제1법칙 (관성) | 1-1 핵심 개념 | 무중력 환경 토글 시 관성 체험 |
| 뉴턴 제2법칙 (F=ma) | 1-1 핵심 개념 | 질량 변경 → 같은 힘에서 가속도 변화 |
| 뉴턴 제3법칙 (작용-반작용) | 1-1 핵심 개념 | 던지기 반동 시각화 |
| 마찰력 | 1-1 핵심 개념 | 공기저항 계수 변수 |
| 중력 | 1-1 핵심 개념 | 중력(g) 변수: 지구/달/목성 |
| 자유낙하 | 1-1 3D 콘텐츠 | 각도=90도, 공기저항 ON/OFF 챌린지 |
| 포물선 운동 | 1-1 3D 콘텐츠 | 각도+중력+공기저항 조합 |

## 엔진 변수

| 변수 | 타입 | 기본값 | 범위 | 설명 |
|------|------|--------|------|------|
| `gravity` | float | 9.8 | 0 ~ 25 | 중력 가속도 (m/s²) |
| `angle` | float | 45 | 0 ~ 90 | 발사 각도 (도) |
| `launchSpeed` | float | 20 | 1 ~ 100 | 발사 속도 (m/s) |
| `mass` | float | 1.0 | 0.1 ~ 100 | 물체 질량 (kg) |
| `drag` | float | 0 | 0 ~ 1 | 공기저항 계수 |
