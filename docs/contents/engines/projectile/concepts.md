# 투사체 엔진 — 핵심 개념 v2

> 소스: 1-1. 운동과 힘 + 3D/XR 필수 확장

## 개념 목록

| 개념 | 챌린지 연결 | 3D/XR 관점 |
|------|-----------|-----------|
| 수평/수직 운동의 독립성 | #1 원숭이와 사냥꾼 | 3D 조준에서 각 축의 독립성 체감 |
| 뉴턴 제1법칙 (관성) | #7 무중력 파괴왕, #8 좌표계 스위치 | 무중력에서 관성만 남는 환경을 3D로 체험 |
| 뉴턴 제2법칙 (F=ma) | #3 커브볼 (마그누스 힘) | 스핀이 만드는 3D 힘의 방향과 크기 |
| 중력과 궤도의 관계 | #2 뉴턴의 대포, #9 달 골프 | 포물선→궤도 전이를 3D 구형 행성 위에서 관찰 |
| 마그누스 효과 | #3 커브볼 | 스핀 축(3D) → 힘 방향(3D) 매핑 |
| 코리올리 힘 | #4 우주 정거장 캐치볼 | 회전 좌표계에서의 가상 힘. 3D 벡터곱 |
| 중력 보조 (Gravity Assist) | #5 중력 슬링샷 | 다중 천체의 3D 중력장에서 에너지 전달 |
| 관성 좌표계와 상대 운동 | #8 좌표계 스위치 | 동일 사건의 복수 3D 시점 비교 |
| 공기저항과 진공 | #9 달 골프 | 대기 유무에 따른 궤적 차이 |
| 시공간 곡률 (입문) | #10 시공간 구슬치기 | 중력을 공간 왜곡으로 시각화 — 3D 격자 |
| 3D 공간 추론 | #6 3D 장애물 사격장 | x,y,z 동시 고려한 궤적 설계 |

## 엔진 변수

| 변수 | 타입 | 기본값 | 범위 | 설명 |
|------|------|--------|------|------|
| `gravity` | float | 9.8 | 0~25 | 중력 가속도 (m/s²) |
| `angle` | vec2 | [45, 0] | [0~90, -180~180] | 발사 각도 (수직각, 수평각) — **3D 확장** |
| `launchSpeed` | float | 20 | 1~100 | 발사 속도 (m/s) |
| `mass` | float | 1.0 | 0.1~100 | 물체 질량 (kg) |
| `drag` | float | 0 | 0~1 | 공기저항 계수 |
| `spinRate` | float | 0 | 0~3000 | 회전 속도 (rpm) — **v2 추가** |
| `spinAxis` | vec3 | [0,1,0] | 단위 벡터 | 스핀 축 방향 — **v2 추가** |
| `rotationRate` | float | 0 | 0~10 | 좌표계 회전 속도 (rpm) — **v2 추가** |
| `gravitySources` | array | [] | - | 다중 중력원 [{mass, position}] — **v2 추가** |
| `obstacles` | array | [] | - | 3D 장애물 [{type, position, size}] — **v2 추가** |
| `targetPosition` | vec3 | null | - | 조준 타깃 위치 — **v2 추가** |
| `targetDropOnLaunch` | bool | false | - | 발사 시 타깃 낙하 여부 — **v2 추가** |
| `environmentPreset` | enum | earth | earth/moon/mars/jupiter/space/station | 환경 프리셋 — **v2 추가** |
