# 투사체 엔진 — 챌린지 시나리오 v2

> **설계 원칙:** 모든 챌린지는 2D에서 불가능하거나 의미가 퇴색하는 것만 포함한다.
> 예측이 틀리는 순간이 가장 재미있다. 반직관적 결과가 호기심을 만든다.
> God Hand 조작이 "슬라이더 조절"이 아닌 "직접 던지고 조준하는" 신체적 경험이어야 한다.

## 시나리오 풀 (10개)

---

### #1 — 원숭이와 사냥꾼 (Monkey & Hunter)

| 항목 | 내용 |
|------|------|
| **핵심 세팅** | 3D 공간에 나무 위 타깃(원숭이 인형). 발사 순간 타깃이 낙하 시작. 거리=15m, 높이=10m, g=9.8 |
| **예측 유형** | placement — 3D 조준점을 선택: "타깃을 맞추려면 어디를 겨냥?" |
| **God Hand** | 새총을 잡고 3D 공간의 한 점을 조준 → 발사. XR: 손으로 새총 당기고 놓기 |
| **WOW 포인트** | 대부분 "타깃 위를 조준해야 한다"고 생각함 → 정답은 타깃을 직접 조준. 공도 떨어지고 타깃도 떨어지므로 같은 비율로 낙하 → 명중. "중력은 모든 걸 똑같이 끌어당긴다!" |
| **3D 필수 이유** | 타깃이 3D 공간의 임의 좌표(x,y,z)에 위치. 수평+수직+깊이를 동시에 고려한 3D 조준 필요. 2D에서는 깊이감이 없어 조준의 공간 감각을 체험 불가 |
| **XR 증폭** | 손으로 직접 새총 조준 → "내 손이 떨린다" 신체적 긴장감. 타깃이 떨어지는 것을 3D 공간에서 실시간으로 추적 |
| **Discover 핵심** | 수평/수직 운동의 독립성. 중력은 발사체와 타깃에 동일하게 작용 |
| **difficulty** | 1 |
| **tags** | `gravity`, `independence-of-motion`, `aiming-3d` |

### #2 — 뉴턴의 대포 (Newton's Cannon)

| 항목 | 내용 |
|------|------|
| **핵심 세팅** | 높은 산꼭대기의 대포. 아래에 둥근 지구가 보임. 발사 속도를 단계적으로 증가: 1km/s → 4km/s → 7.9km/s → 11.2km/s |
| **예측 유형** | trajectory — "속도를 올리면 포탄 궤적이 어떻게 변할까?" 4개 속도 각각의 3D 궤적 예측 |
| **God Hand** | 대포의 화력 레버를 드래그로 당겨 발사 강도 조절 → 발사. XR: 대포 옆에 서서 레버 조작 |
| **WOW 포인트** | 속도를 계속 올리면: 가까이 떨어짐 → 멀리 떨어짐 → 지구 반대편 → **떨어지는데 빗나간다 = 궤도!** → 더 빠르면 지구를 벗어남. "떨어지면서 빗나가는 것이 궤도의 본질이다!" |
| **3D 필수 이유** | 둥근 행성 주위를 도는 궤도는 본질적으로 3D. 산꼭대기에서 구형 지구를 내려다보며 포탄이 지구를 감싸 도는 것은 3D에서만 체감 가능 |
| **XR 증폭** | 산꼭대기에 서서 발 아래로 구형 지구. 포탄이 지구를 감싸 돌아오는 것을 직접 목격 |
| **Discover 핵심** | 포물선 운동과 궤도 운동의 연결. v < v_orbit → 추락, v = v_orbit → 원궤도, v > v_escape → 탈출 |
| **difficulty** | 3 |
| **tags** | `gravity`, `orbital-mechanics`, `escape-velocity` |

| **예측 유형** | trajectory — "이 스핀을 넣으면 공이 3D에서 어떻게 휘어질까?" 3D 궤적 그리기 |

| **WOW 포인트** | 백스핀이 공을 "떠오르게" 만든다 (마그누스 힘이 중력에 반함). 사이드스핀은 1미터 이상 횡방향으로 휜다. "회전이 공기의 압력차를 만들어 힘을 만든다!" |

| **XR 증폭** | 투수 시점에서 공을 던지며 스핀을 부여. 포수 시점에서 공이 날아오는 것을 보면 "왜 타자가 못 치는지" 체감 |
| **Discover 핵심** | 마그누스 효과 — 회전체의 공기압 차이가 횡방향 힘을 만든다. 스포츠(축구 프리킥, 탁구 스핀)과 연결 |
| **difficulty** | 3 |
| **tags** | `magnus-effect`, `spin`, `air-pressure`, `3d-trajectory` |

## 예측 유형 분포

| 유형 | 시나리오 | 수 | 비율 |
|------|---------|---|------|
| trajectory (3D 궤적 그리기) | #2, #3, #4, #5, #6, #9, #10 | 7 | 70% |
| placement (3D 지점 선택) | #1, #7 | 2 | 20% |
| pattern (선택지) | #8 | 1 | 10% |

> trajectory가 여전히 주력이지만, 기존과 달리 **모든 궤적이 3D 공간에서 전개**되어 "포물선 그리기"의 반복이 아님.
> 각 시나리오마다 궤적의 성격이 다름: 코리올리 곡선, 마그누스 커브, 중력 슬링샷 나선, 시공간 곡률 경로 등.

## 엔진 확장 변수 (v2)

기존 변수(`gravity`, `angle`, `launchSpeed`, `mass`, `drag`)에 추가:

| 변수 | 타입 | 범위 | 용도 |
|------|------|------|------|
| `spinRate` | float | 0~3000 (rpm) | 마그누스 효과 (#3) |
| `spinAxis` | vec3 | 단위 벡터 | 스핀 방향 (#3) |
| `rotationRate` | float | 0~10 (rpm) | 회전 좌표계 (#4) |
| `gravitySources` | array<{mass, position}> | - | 다중 천체 (#5, #10) |
| `obstacles` | array<{type, position, size}> | - | 장애물 코스 (#6) |
| `targetPosition` | vec3 | - | 조준 챌린지 (#1, #6) |
| `targetDropOnLaunch` | bool | - | 발사 시 타깃 낙하 (#1) |
| `environmentPreset` | enum | earth/moon/jupiter/mars/space/station | 환경 프리셋 (#2, #4, #7, #9) |

## 온보딩 챌린지

**#1 원숭이와 사냥꾼**의 단순 버전을 온보딩으로 사용:
- 타깃 고정(낙하 없음), 거리 짧게 → "3D에서 타깃 맞추기"만 체험
- 성공 → 타깃 낙하 ON → 본격 챌린지 시작
- 30초 안에 코어 루프(조준 → 발사 → 비교) 1회 완주
