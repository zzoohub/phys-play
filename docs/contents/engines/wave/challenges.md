# 파동 엔진 — 챌린지 시나리오 v2

> **설계 원칙:** 파동이야말로 3D의 최대 수혜자다. 실제 세계의 파동은 3D로 퍼지지만, 교과서에서 항상 2D로 압축한다.
> "직접 걸어다니며 소리가 달라지는" 체험은 2D에서 절대 불가능하다.
> 파동의 "보이지 않는 것을 보이게 만드는" 시각화가 3D에서 가장 강력하다.

## 시나리오 풀 (8개)

### #1 — 소리의 방 (Sound Room)

| 항목 | 내용 |
|------|------|
| **핵심 세팅** | 직사각형 3D 방(6m×4m×3m). 한쪽 벽에 스피커(주파수 조절 가능). 음파가 3D로 시각화되어 퍼짐. 벽에서 반사 |
| **예측 유형** | placement — "이 방에서 소리가 가장 작은 '죽은 지점'은 어디?" 3D 공간에 마커 배치 |
| **God Hand** | 스피커 위치와 주파수 조절. 마커를 3D 공간의 원하는 위치에 배치. XR: 방 안을 직접 걸어다니며 소리 변화를 듣고, "여기!" 하며 마커를 놓기 |
| **WOW 포인트** | 10cm만 움직여도 소리가 극적으로 변한다! 큰 소리 → 죽은 지점 → 다시 큰 소리. "소리에도 줄무늬가 있다!" 방 안의 정상파 노드가 3D로 존재한다는 사실. "콘서트홀 설계자가 이것을 고려해서 좌석을 배치한다!" |
| **3D 필수 이유** | 정상파 노드(소리가 상쇄되는 지점)는 3D 공간에 입체적으로 분포. 바닥 높이에서는 소리가 들리지만 서 있는 높이에서는 안 들릴 수 있다. 2D에서는 이 "높이에 따른 차이"를 체험 불가 |
| **XR 증폭** | 실제로 방 안을 걸어다니며 소리의 세기가 변하는 것을 몸으로 체험. 손을 뻗어 다른 높이에서의 소리 차이를 느끼기. 음파 시각화가 내 몸을 관통하는 감각 |
| **Discover 핵심** | 정상파와 노드/배. 벽 반사에 의한 3D 간섭. 방의 크기와 파장의 관계가 "방 음향학"을 결정 |
| **difficulty** | 2 |
| **tags** | `standing-wave`, `node`, `antinode`, `room-acoustics`, `3d-interference` |

| **예측 유형** | placement — "안티 스피커를 어디에 놓으면 이 영역이 조용해질까?" 3D 공간에 스피커 배치 |
| **God Hand** | 안티 스피커를 잡아 3D 공간에 배치. 실시간으로 간섭 패턴이 변하며 조용한 영역이 시각화됨. XR: 스피커를 들고 돌아다니며 최적 위치 탐색, 조용한 영역에 서서 효과 확인 |

| **3D 필수 이유** | 안티 스피커의 3D 위치에 따라 상쇄 영역의 3D 형태가 달라짐. "높이"를 바꾸면 상쇄 영역이 위아래로 이동. 2D에서는 평면적 배치만 가능 |
| **XR 증폭** | 실제로 소음이 들리는 영역 → 안티 스피커 배치 → 고요해지는 것을 듣는 신체적 체험. "여기서 소리가 사라졌어!" |
| **Discover 핵심** | 상쇄 간섭의 실용적 적용. 경로차 = (n+½)λ에서 상쇄. ANC 기술의 물리학. "소리를 더해서 소리를 빼는" 역설 |
| **difficulty** | 3 |

### #3 — 소닉 붐 (Sonic Boom)

| 항목 | 내용 |
|------|------|
| **핵심 세팅** | 공기 중을 이동하는 물체. 속도 슬라이더: 0 → 마하 0.5 → 마하 1.0 → 마하 2.0. 음파 파면이 3D로 시각화 |
| **예측 유형** | pattern — "물체가 마하 1을 넘으면 파면이 어떤 모양이 될까?" 3D 파면 형태 3개 중 선택 (동심원/평면벽/원뿔) |
| **God Hand** | 속도 슬라이더를 드래그하며 파면 변화를 실시간 관찰. 물체 경로를 3D로 변경 가능(직선/곡선/급선회). XR: 물체와 나란히 "비행"하며 콘이 형성되는 것을 관찰 |
| **WOW 포인트** | 아음속: 정상적 동심원. 마하 1 돌파 순간: 파면이 쌓이다가 **충격파 벽**이 형성! 초음속: 아름다운 마하 콘이 뒤로 펼쳐짐. "배에서 V자 물결이 생기는 것과 같은 원리!" 전투기 소닉 붐의 비밀. 속도를 올리면 콘 각도가 좁아지는 것도 시각적으로 확인 |
| **3D 필수 이유** | 마하 콘은 3D 원뿔이다. 2D에서는 V자 선으로 보이지만, 실제로는 물체를 중심축으로 한 3D 원뿔. 콘 내부/외부의 소리 차이도 3D 공간에서만 체험 가능 |
| **XR 증폭** | 초음속 물체와 나란히 비행. 마하 콘 내부에 있으면 조용하다가, 콘 바깥으로 나가는 순간 "빵!" 소닉 붐을 듣는 체험 |
| **Discover 핵심** | 도플러 효과의 극한: 음원 속도 ≥ 음속. 마하 수와 콘 각도 관계: sin θ = v_sound/v_object. 충격파와 소닉 붐 |
| **difficulty** | 3 |
| **tags** | `sonic-boom`, `mach-cone`, `shock-wave`, `doppler-extreme`, `supersonic` |

### #4 — 와인잔 깨기 (Resonance Shatter)

| 항목 | 내용 |
|------|------|
| **핵심 세팅** | 와인잔 1개(고유 진동수 ~550Hz). 주파수 조절 가능한 스피커. 와인잔의 3D 변형이 실시간 시각화(과장된 슬로모션으로) |
| **예측 유형** | binary — "이 주파수에서 잔이 깨질까?" (주파수를 바꿔가며 여러 번) → 최종: "정확히 어떤 주파수에서 깨지는가?" 슬라이더로 탐색 |
| **God Hand** | 주파수 슬라이더를 돌려 주파수 변경. 와인잔이 진동하는 것을 확인하며 공명점을 "사냥". XR: 마이크에 대고 직접 소리를 내거나(허밍), 슬라이더로 주파수 조절 |
| **WOW 포인트** | 공명점에 가까워질수록 잔이 **젤리처럼 흔들린다** (3D 변형 과장 시각화). 타원 → 팔자 → 점점 심하게 → **산산조각!** 공명의 파괴력을 시각적으로 체험. "오페라 가수가 고음으로 잔을 깰 수 있다는 게 사실이다!" |
| **3D 필수 이유** | 와인잔의 진동 모드는 3D — 원형 단면이 타원으로, 다시 수직으로 타원으로 진동. 이 3D 변형은 2D에서 표현 불가. 위에서, 옆에서 보면 진동 패턴이 다르게 보임 |
| **XR 증폭** | 바로 앞에서 와인잔이 흔들리는 것을 보고, 깨지는 순간 파편이 사방으로 튀는 것을 체감. 공명점을 찾는 "보물찾기" 감각 |
| **Discover 핵심** | 공명(resonance) — 외부 진동수가 고유 진동수와 일치할 때 진폭이 폭발적으로 증가. 타코마 다리 붕괴, 군대 행진이 다리를 무너뜨리는 이야기 |
| **difficulty** | 1 |
| **tags** | `resonance`, `natural-frequency`, `vibration-mode`, `destruction`, `3d-deformation` |

### #5 — 지구 내부 지진파 (Earthquake Inside)

| 항목 | 내용 |
|------|------|
| **핵심 세팅** | 투명한 지구 단면(3D 구체). 내부 레이어: 지각, 맨틀, 외핵(액체), 내핵(고체). 지진 발생점 설정 가능. P파(종파)와 S파(횡파) 동시 시각화 |
| **예측 유형** | placement — "지진이 여기서 발생하면, 지구 반대편에서 P파와 S파가 각각 어디에 도착할까?" 3D 구체 표면에 도착점 마커 배치 |
| **God Hand** | 지구 표면의 아무 점을 터치해 지진 발생 → 내부로 파동 전파. 지구를 회전/확대하여 내부를 관찰. XR: 투명 지구 내부에 서서 파동이 내 주변을 지나가는 것을 관찰 |
| **WOW 포인트** | P파는 모든 매질을 통과하지만, **S파는 액체 외핵을 통과하지 못한다!** 그래서 지구 반대편에 "S파 그림자 영역(shadow zone)"이 생긴다! "이것이 과학자들이 지구 내핵이 액체라는 것을 알아낸 방법이다!" — 파동의 성질로 보이지 않는 내부를 알아낸 인류의 지혜 |
| **3D 필수 이유** | 지구는 3D 구체. 파동이 3D 구체 내부를 곡선 경로로 전파(밀도 차이에 의한 굴절). 그림자 영역도 3D 원뿔형. 2D 단면도에서는 구체의 3D 전파를 표현 불가 |
| **XR 증폭** | 투명 지구 내부에 서서 P파와 S파가 나를 관통하는 것을 체험. 외핵에서 S파가 "차단"되는 것을 내부에서 직접 목격 |
| **Discover 핵심** | 종파(P파)와 횡파(S파)의 차이. 횡파는 액체를 통과 못 함 (전단 응력 전달 불가). 지진학으로 지구 내부 구조를 밝힌 과학사 |
| **difficulty** | 4 |
| **tags** | `p-wave`, `s-wave`, `seismology`, `earth-interior`, `wave-propagation` |

### #6 — 3D 도플러 비행 (3D Doppler Flight)

| 항목 | 내용 |
|------|------|
| **핵심 세팅** | 관찰자(나) 고정. 사이렌을 울리는 드론이 3D 경로로 비행: 직선 통과, 원형 선회, 나선 하강, 급접근+급선회 등 |
| **예측 유형** | pattern — "이 3D 경로로 비행할 때 내가 듣는 피치 변화 곡선은?" 피치-시간 그래프 3개 중 선택 |
| **God Hand** | 드론의 3D 비행 경로를 직접 그리기(웨이포인트 배치). 실행 후 드론 비행 + 실시간 피치 변화 시각화/청각화. XR: 드론이 내 주위를 3D로 비행하며 실시간 도플러 변화를 듣기 |
| **WOW 포인트** | 단순 "높아졌다 낮아졌다"가 아님! 3D 경로에서는 도플러 효과가 훨씬 복잡: 원형 선회 시 주기적 피치 변화, 나선 접근 시 점진적 피치 상승+진폭 증가, 측면 통과 시 급격한 피치 드롭. "구급차가 지나가면 소리가 변하는 이유가 이거였어!" — 일상 현상의 물리학 |
| **3D 필수 이유** | 도플러 변위량은 소스와 관찰자의 상대 속도(3D 벡터)의 시선 방향 성분에 의존. 2D에서는 "왼→오" 통과만 가능하지만, 3D에서는 위/아래/나선/선회 등 무한한 경로에서 각각 다른 도플러 패턴 |
| **XR 증폭** | 실제로 내 주변을 3D로 비행하는 드론의 사이렌 피치 변화를 공간 오디오로 듣기. 고개를 돌려 드론을 추적하며 피치 변화를 체감 |
| **Discover 핵심** | 도플러 효과: f' = f(v±v_observer)/(v∓v_source). 시선 방향 속도 성분만 영향. 적색편이(우주 팽창) + 경찰 스피드건과의 연결 |
| **difficulty** | 3 |
| **tags** | `doppler`, `frequency-shift`, `3d-path`, `relative-velocity`, `everyday-physics` |

### #7 — 속삭임의 방 (Whispering Gallery)

| 항목 | 내용 |
|------|------|
| **핵심 세팅** | 타원형 돔 공간(장축 30m). 세인트 폴 대성당/미국 국회의사당의 속삭임 갤러리 재현. 한쪽 초점에 화자, 반대쪽에 청자 |
| **예측 유형** | placement — "여기서 속삭이면, 30m 건너편 어디에서 또렷하게 들을 수 있을까?" 3D 공간에 청취점 마커 배치 |
| **God Hand** | 화자 위치를 이동시키며 소리 경로 시각화. 청자 위치를 이동시키며 소리 세기 확인. XR: 돔 안을 걸어다니며 속삭임이 들리는 "달콤한 지점(sweet spot)"을 탐색 |
| **WOW 포인트** | 30미터 건너의 속삭임이 바로 옆에서 말하는 것처럼 들린다! 하지만 중간 지점에서는 거의 안 들린다. "타원의 수학이 소리를 모은다!" 돔 벽면의 반사 경로가 시각화되면 — 모든 반사 경로가 한 점으로 수렴하는 것이 기하학적으로 아름답다 |
| **3D 필수 이유** | 타원형 돔의 3D 반사 기하학. 소리가 돔 곡면을 따라 3D로 전파. 2D에서는 타원의 2D 반사만 보이지만, 실제 돔은 3D 회전 타원체이므로 위아래에서도 반사 경로가 존재 |
| **XR 증폭** | 거대한 돔 내부에 서 있는 공간감. 속삭임이 벽을 타고 돌아오는 것을 공간 오디오로 듣기. 한쪽 초점에서 속삭이고 반대쪽으로 걸어가서 "들리는 순간"의 놀라움 |
| **Discover 핵심** | 타원의 초점 성질: 한 초점에서 나온 파동은 벽 반사 후 반드시 다른 초점에 모인다. 반사의 법칙 + 기하학. 건축 음향학의 물리 |
| **difficulty** | 2 |
| **tags** | `reflection`, `ellipse-focus`, `architectural-acoustics`, `geometry`, `3d-sound` |

### #8 — 악기의 물리학 (Musical Instrument Physics)
