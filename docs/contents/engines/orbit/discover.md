## 설명 소재

| 개념 | 직관적 비유 (Level 1) | 개념 설명 (Level 2) |
|------|----------------------|--------------------|
| 케플러 법칙 | "안쪽 트랙 선수가 더 빨리 도는 것 + 태양에 가까울 때 더 빨라지는 것" | T²∝a³, 면적속도 보존, 타원의 장반축이 주기를 결정 |
| 탈출 속도 | "공을 더 세게 던지면 더 멀리 — 어느 순간 돌아오지 않는다" | v_esc = √(2GM/r). 원궤도 → 타원 → 포물선 → 쌍곡선의 연속 스펙트럼 |
| 일식 조건 | "정확히 같은 크기의 동전으로 전등을 가리기" | 달 궤도면 5.14° 기울기 + 교점 근처의 삭(new moon) = 드문 조건의 결합 |
| 조석력 | "달이 지구를 양쪽으로 '잡아늘이는' 것" | 기조력 = 거리에 따른 중력 차이. 가까운 쪽은 더 끌리고, 먼 쪽은 덜 끌린다 |
| 로슈 한계 | "너무 가까이 가면 달이 찢어진다!" | 조석력 > 자체 중력 → 천체 파괴. 토성의 고리는 로슈 한계 안의 잔해 |
| 라그랑주 점 | "두 코끼리 사이에서 균형 잡는 개미" | 두 천체의 중력 + 원심력이 균형을 이루는 5개 특수 지점. L4/L5만 안정 |
| 삼체 문제 | "세 사람이 서로 줄로 연결되어 달리기 — 예측 불가!" | 해석적 해 없음. 초기 조건의 극미한 차이 → 완전히 다른 결과 (카오스) |
| 쌍성계 | "두 스케이터가 손잡고 도는 것 — 무거운 쪽은 덜 움직인다" | 공통 질량 중심 주위 공전. 질량비에 따라 중심 이동 |
| 은하 충돌 | "두 벌떼가 서로 통과하는 것 — 벌끼리 부딪칠 확률은 거의 0" | 별 사이 평균 거리 ≫ 별 크기. 중력 상호작용이 구조를 바꾸지만 직접 충돌은 극히 드물다 |
| 스케일 | "원자에서 우주까지 10^36배 — 매 단계마다 전혀 다른 세계" | 핵력(10⁻¹⁵m) → 전자기력(10⁻⁹m) → 중력(10⁶m~) 이 각 스케일의 지배적 힘 |
