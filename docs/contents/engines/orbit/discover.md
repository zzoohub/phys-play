## 설명 소재

| 개념 | 직관적 비유 (Level 1) | 개념 설명 (Level 2) |
|------|----------------------|--------------------|
| 공전 | "빙글빙글 도는 놀이기구에서 줄이 끊어지면 바깥으로 날아가는 것" | 중력이 구심력 역할을 해서 천체를 궤도에 묶어둔다 |
| 케플러 법칙 | "안쪽 트랙 선수가 더 빨리 도는 것처럼, 태양에 가까운 행성이 더 빨리 돈다" | 궤도 주기의 제곱은 궤도 반지름의 세제곱에 비례 (T² ∝ r³) |
| 일식 | "손전등과 눈 사이에 동전을 정확히 끼워 넣어야 빛이 가려지는 것" | 달의 궤도 기울기 때문에 달이 태양을 정확히 가리는 경우는 드물다 |
| 조석 | "달이 가까운 쪽 바다를 더 세게 잡아당기는 것" | 달의 중력 차이(기조력)가 지구 양쪽에 밀물을 만든다 |
| 탈출 속도 | "공을 충분히 세게 던지면 지구로 돌아오지 않는 것" | 중력을 이겨내려면 특정 속도 이상이 필요하다 (v_esc = √(2GM/r)) |
| 적색편이 | "멀어지는 구급차 소리가 낮아지는 것의 빛 버전" | 은하가 멀어지면 빛의 파장이 길어져(붉어져) 보인다 |

## 개념 연결 (크로스 참조)

- **중력** ← 투사체 엔진: "지구에서의 g=9.8은 만유인력의 특수한 경우"
- **적색편이** ← 파동 엔진: "파동에서 배운 도플러가 우주 팽창의 증거가 된다"
