# 궤도 엔진 — 핵심 개념 v2

> 소스: 3-1. 지구와 달 + 3-2. 태양계 + 3-3. 별의 일생 (중력 부분) + 3-4. 빛과 스펙트럼 (적색편이) + 3-5. 은하 (중력 구조)
> v2: 3D/XR 관점 추가, 엔진 변수 확장

## 개념 목록

| 개념 | 챌린지 연결 | 3D/XR 관점 |
|------|-----------|-----------|
| 케플러 법칙 (타원궤도, 면적속도, 주기) | #1 태양계 건축가 | 3D 타원 궤도를 직접 만들며 법칙 체험 |
| 궤도면 기울기와 일식 조건 | #2 일식 엔지니어 | 5.14° 기울기가 일식 발생을 결정하는 3D 기하학 |
| 조석력과 기조력 | #3 조석력 실험실 | 3D 기조력 벡터장과 지구 변형 시각화 |
| 라그랑주 점과 3체 제한 문제 | #4 라그랑주 포인트 헌트 | 3D 유효 포텐셜 지형의 안장점 탐색 |
| 삼체 문제와 카오스 | #5 삼체 카오스 | 3D 궤적의 카오스적 발산. 초기 조건 민감성 |
| 쌍성계와 질량 중심 | #6 쌍성 댄스 | 공통 질량 중심을 3D에서 찾고 궤도면 기울기에 따른 관측 차이 |
| 은하 충돌과 조석 상호작용 | #7 은하 충돌 | N-body 시뮬레이션의 3D 접근 각도에 따른 결과 차이 |
| 우주의 계층 구조와 스케일 | #8 스케일 계단 | 10^36배 스케일 변화를 3D 줌으로 체험 |
| 탈출 속도와 궤도 전이 | #1 태양계 건축가 | 던지는 속도에 따른 궤도 유형(원/타원/포물선/쌍곡선) 변화 |
| 로슈 한계 | #3 조석력 실험실 | 천체가 조석력으로 파괴되는 3D 과정 시각화 |
