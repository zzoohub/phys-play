# 분자 엔진 — 챌린지 시나리오 v2

> 소스: 2-1. 원자와 결합 + 2-2. 분자의 형태 + 2-3. 화학반응 + 2-4. 생체분자
> v2 원칙: Must 3D/XR · 게이미피케이션 핵심 · WOW Point 필수

## v2 설계 철학

| 원칙 | 적용 |
|------|------|
| Must 3D | 분자는 본질적으로 3D 구조물. 2D 그림으로는 결합각, 입체화학, 단백질 접힘을 체감 불가 |
| WOW Point | "같은 원자인데 배열만 다르면 완전히 다른 물질?" — 3D 구조가 기능을 결정하는 놀라움 |
| God Hand | 원자를 손으로 잡고 결합, 분자를 회전/뒤집기, 약물을 도킹 포켓에 끼우기 |
| XR 증폭 | 분자 크기로 축소되어 내부를 걸어다니며 탐험. 손으로 전자구름을 만지는 촉각 |

## 시나리오 풀

### #1 분자 조각가 (Molecular Sculptor)

| 항목 | 내용 |
|------|------|
| **컨셉** | 원자들을 3D 공간에서 손으로 조립하여 목표 분자를 만든다 |
| **Must 3D** | VSEPR 이론은 3D 전자쌍 반발. 2D에서는 109.5°와 120°의 차이를 체감 불가 |
| **변수 세팅** | `targetMolecule=H₂O`, `availableAtoms=[H,H,O]`, `showElectronPairs=true` |
| **예측 (placement)** | "이 원자들을 결합시키면 어떤 3D 모양이 될까? 직접 배치해봐" |
| **God Hand** | 원자를 손으로 잡아 다른 원자에 가까이 가져가면 전자구름이 반응. 결합각을 직접 느낀다 |
| **WOW Point** | 물 분자를 직선으로 배치하면 전자쌍 반발로 "팍!" 하고 104.5°로 꺾인다 |
| **XR 증폭** | 분자 크기로 축소되어 전자구름 안에 서서 반발력을 체감 |
| **Discover 핵심** | VSEPR 이론, 비공유 전자쌍의 영향, 결합각 |
| **난이도** | 1 |

### #2 약물 도킹 퍼즐 (Drug Docking)

| 항목 | 내용 |
|------|------|
| **컨셉** | 효소의 3D 활성 부위에 약물 분자를 올바른 방향으로 끼워넣는 퍼즐 |
| **Must 3D** | 약물-효소 결합은 3D 열쇠-자물쇠. 2D 그림으로는 입체적 적합성을 체감 불가 |
| **변수 세팅** | `enzyme=lysozyme`, `drugCandidates=[A,B,C]`, `bindingPocket=3Dmesh` |
| **예측 (placement)** | "세 약물 중 어떤 것이 이 포켓에 맞을까? 직접 끼워봐" |
| **God Hand** | 약물 분자를 손으로 잡고 회전시켜 효소 포켓에 맞춰본다. 수소결합 위치가 하이라이트 |
| **WOW Point** | 거울상 이성질체(카이랄 분자) — 왼손/오른손 장갑처럼 3D 형태만 다른데 하나는 약, 하나는 독 |
| **XR 증폭** | 효소 내부로 들어가서 포켓 벽면의 전하 분포를 보며 도킹. 결합 시 "찰칵" 촉각 피드백 |
| **Discover 핵심** | 입체화학, 카이랄성, 효소-기질 특이성, 약물 설계 |
| **난이도** | 4 |

### #3 극성 배틀 (Polarity Battle)

| 항목 | 내용 |
|------|------|
| **컨셉** | 3D 공간에 극성/비극성 분자와 용매를 배치하여 "녹는다/안 녹는다"를 예측 |
| **Must 3D** | CO₂(직선형=비극성)와 H₂O(꺾인형=극성)의 차이는 3D 분자 대칭에서 온다 |
| **변수 세팅** | `molecules=[CO₂, H₂O, NaCl, oil]`, `solvent=water`, `showDipoles=true` |
| **예측 (pattern)** | "이 분자들 중 물에 녹는 것과 녹지 않는 것을 분류해봐" |
| **God Hand** | 분자를 물 탱크에 던져넣는다. 극성 분자는 물 분자들이 달라붙어 용해, 비극성은 튕겨나옴 |
| **WOW Point** | CO₂와 H₂O는 같은 3개 원자인데 — 대칭 하나로 극성이 반대 → 물에 녹는지가 결정된다 |
| **XR 증폭** | 물 분자 사이에 서서 극성 분자가 수소결합으로 끌려들어가는 과정을 관찰 |
| **Discover 핵심** | 분자 대칭, 극성/비극성, 용해 원리, "같은 것끼리 녹인다" |
| **난이도** | 2 |

### #4 반응 극장 (Reaction Theater)

| 항목 | 내용 |
|------|------|
| **컨셉** | 분자들이 3D 공간에서 충돌하는 것을 슬로모션으로 관찰. 활성화 에너지 언덕을 넘는 과정 |
| **Must 3D** | 분자 충돌의 "올바른 방향" — 3D에서만 배향(orientation) 효과를 체감 가능 |
| **변수 세팅** | `reactants=[H₂,I₂]`, `temperature=500`, `catalyst=false`, `showEnergyLandscape=true` |
| **예측 (binary)** | "현재 온도에서 이 분자들이 반응할까?" |
| **God Hand** | 온도 슬라이더 대신 — 공간 전체를 "흔들어서" 분자 운동 속도를 높인다. 촉매를 직접 배치 |
| **WOW Point** | 3D 에너지 지형 위에서 반응물이 '산'을 넘는 과정. 촉매를 놓으면 산에 '터널'이 뚫린다 |
| **XR 증폭** | 분자 크기로 축소되어 충돌 현장에 서 있다. 활성화 에너지 '산'을 3D 지형으로 걷는다 |
| **Discover 핵심** | 활성화 에너지, 반응 속도론, 촉매 작용, 배향 효과 |
| **난이도** | 3 |

### #5 단백질 접기 레이스 (Protein Folding Race)

| 항목 | 내용 |
|------|------|
| **컨셉** | 아미노산 사슬을 3D로 접어 목표 구조에 최대한 가깝게 만드는 경쟁 |
| **Must 3D** | 단백질 접힘은 본질적으로 3D. α-헬릭스, β-시트의 공간적 배열 |
| **변수 세팅** | `sequence=[Ala,Gly,Val,Leu,...]`, `targetStructure=3Dmesh`, `hydrophobic=highlighted` |
| **예측 (placement)** | "이 아미노산 사슬이 물 속에서 어떻게 접힐까? 소수성 잔기를 안쪽으로 배치해봐" |
| **God Hand** | 아미노산 사슬의 각 관절을 손으로 접는다. 소수성 부분은 물을 피하려 안쪽으로 모인다 |
| **WOW Point** | "놓으면" 물리 시뮬레이션이 돌면서 자동으로 접힌다 — 자신의 예측과 실제 접힘 비교 |
| **XR 증폭** | 단백질 내부를 걸어다니며 소수성 코어와 친수성 표면의 차이를 체감 |
| **Discover 핵심** | 소수성 상호작용, 수소결합, 단백질 구조(1차→4차), 레빈탈 역설 |
| **난이도** | 5 |

### #6 DNA 이중나선 탐험 (DNA Helix Walk)

| 항목 | 내용 |
|------|------|
| **컨셉** | DNA 이중나선 내부를 걸어다니며 상보적 염기쌍을 맞추고, 복제/전사 과정을 체험 |
| **Must 3D** | DNA는 나선형 3D 구조. 주홈/부홈의 3D 형태가 단백질 결합에 결정적 |
| **변수 세팅** | `sequence=[A,T,G,C,T,A,G,C]`, `mode=replication`, `showHydrogenBonds=true` |
| **예측 (pattern)** | "빈 칸의 상보적 염기를 채워넣어 이중나선을 완성해봐" |
| **God Hand** | 염기를 손으로 잡아 상대 가닥에 맞춰본다. 맞으면 수소결합이 "척!" 하고 형성 |
| **WOW Point** | DNA 나선 안에 서서 올려다보면 끝없이 꼬인 사다리. 복제 포크가 지나가는 것을 목격 |
| **XR 증폭** | 나선 내부를 걸어다니며 A-T(2개 수소결합) vs G-C(3개 수소결합)의 차이를 촉각으로 체감 |
| **Discover 핵심** | 상보적 염기쌍, 수소결합, DNA 복제 메커니즘, 유전 정보 저장 |
| **난이도** | 2 |

### #7 르샤틀리에의 시소 (Le Chatelier's Seesaw)

| 항목 | 내용 |
|------|------|
| **컨셉** | 화학 평형 상태의 3D 반응기에 온도/압력/농도를 변화시켜 이동 방향을 예측 |
| **Must 3D** | 3D 반응 용기 안에서 분자 수 변화를 실시간으로 보며 평형 이동을 체감 |
| **변수 세팅** | `reaction=N₂+3H₂⇌2NH₃`, `temperature=500`, `pressure=200`, `perturbation=pressure_up` |
| **예측 (pattern)** | "압력을 높이면 평형이 어느 쪽으로 이동할까?" |
| **God Hand** | 3D 반응기의 피스톤을 손으로 누른다(압력 증가). 분자 색상 변화로 평형 이동을 실시간 관찰 |
| **WOW Point** | 피스톤을 누를수록 NH₃(파란색)가 더 많아진다 — 분자 수가 적은 쪽으로 이동하는 것을 시각적으로 확인 |
| **XR 증폭** | 반응기 안에 서서 분자들이 충돌하고 결합/분해되는 것을 관찰. 피스톤 압축 시 밀도 변화 체감 |
| **Discover 핵심** | 르샤틀리에 원리, 화학 평형, 압력/온도/농도 효과 |
| **난이도** | 3 |

### #8 결정 격자 건축가 (Crystal Lattice Builder)

| 항목 | 내용 |
|------|------|
| **컨셉** | 이온/원자를 3D 공간에 배열하여 결정 구조를 만든다. 배열이 물성을 결정 |
| **Must 3D** | 결정 구조는 3D 단위셀의 반복. FCC/BCC/HCP는 2D 그림으로 구별 불가 |
| **변수 세팅** | `atoms=[Na,Cl]`, `latticeType=targetHidden`, `targetProperty=hardness` |
| **예측 (placement)** | "Na⁺와 Cl⁻를 배열하여 가장 단단한 결정을 만들어봐" |
| **God Hand** | 이온을 3D 격자점에 하나하나 배치. 전하 반발/인력이 힘 피드백으로 느껴진다 |
| **WOW Point** | 같은 원소도 배열만 바꾸면 — 흑연(연필심) vs 다이아몬드(세계에서 가장 단단한 것) |
| **XR 증폭** | 격자 내부를 걸어다니며 반복 패턴을 체감. 단위셀 하나를 "복제"하여 결정 성장 관찰 |
| **Discover 핵심** | 이온결합, 결정 구조, 단위셀, 물성과 구조의 관계, 동소체 |
| **난이도** | 3 |
