# 분자 엔진 — 핵심 개념

> 소스: 2-1. 원자와 결합 + 2-2. 분자의 형태 + 2-3. 화학반응 + 2-4. 생체분자 (구조 부분)

## 개념 목록

| 개념 | 출처 | 엔진 변수와의 연결 |
|------|------|-------------------|
| 원자 구성 (양성자, 중성자, 전자) | 2-1 | 원소 종류 변수 (원자번호) |
| 전자 배치와 원자가 전자 | 2-1 | 결합 가능성 결정 |
| 공유결합, 이온결합, 금속결합 | 2-1 | 결합 유형 변수 |
| VSEPR 이론과 분자 기하학 | 2-2 | 전자쌍 수 → 분자 모양 자동 결정 |
| 극성과 비극성 | 2-2 | 극성 화살표 오버레이 |
| 분자간 힘 (수소결합, 반데르발스) | 2-2 | 분자간 거리/온도 변수 |
| 반응 메커니즘 | 2-3 | 온도, 농도 변수 → 충돌 확률 |
| 활성화 에너지와 촉매 | 2-3 | 촉매 ON/OFF, 에너지 장벽 높이 |
| 화학 평형 | 2-3 | 온도/압력/농도 변수 (르샤틀리에) |
| 단백질 접힘 | 2-4 | 아미노산 시퀀스 → 3D 구조 |
| DNA 이중나선 | 2-4 | 염기쌍 매칭 변수 |

## 엔진 변수

| 변수 | 타입 | 기본값 | 범위 | 설명 |
|------|------|--------|------|------|
| `elements` | string[] | — | 주기율표 원소 | 조합할 원소 목록 |
| `electronPairs` | int | auto | 1 ~ 6 | 전자쌍 수 (VSEPR) |
| `bondType` | enum | auto | covalent / ionic / metallic | 결합 유형 |
| `temperature` | float | 298 | 100 ~ 5000 | 온도 (K) |
| `pressure` | float | 1.0 | 0.1 ~ 100 | 압력 (atm) |
| `concentration` | float | 1.0 | 0.01 ~ 10 | 농도 (mol/L) |
| `catalyst` | bool | false | true / false | 촉매 존재 여부 |
| `sequence` | string | — | A/T/G/C | DNA/RNA 서열 |
