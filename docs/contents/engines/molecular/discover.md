## 설명 소재

| 개념 | 직관적 비유 (Level 1) | 개념 설명 (Level 2) |
|------|----------------------|--------------------|
| VSEPR | "풍선 4개를 묶으면 사면체, 3개면 삼각형 — 전자쌍도 최대한 멀어지려 한다" | 전자쌍 반발이 분자의 3D 모양을 결정한다. 비공유 전자쌍은 공유 전자쌍보다 더 세게 밀어낸다 |
| 카이랄성 | "왼손과 오른손 — 똑같은 구조인데 겹쳐지지 않는다" | 거울상 이성질체는 3D 공간에서 겹칠 수 없다. 한쪽은 약, 한쪽은 독이 될 수 있다 (탈리도마이드) |
| 극성 | "한쪽이 더 힘이 센 줄다리기 — 하지만 CO₂는 양쪽이 같아서 상쇄" | 결합의 극성 + 분자의 3D 대칭이 결합하여 분자 전체의 극성을 결정 |
| 활성화 에너지 | "3D 산맥 위의 고개 — 가장 낮은 고개를 찾아야 반응이 일어난다" | 3D 에너지 지형에서 반응물→생성물 경로의 최저 에너지 안장점 |
| 촉매 | "산에 터널을 뚫어 지름길을 만드는 것" | 활성화 에너지를 낮춰 반응 속도를 높인다. 촉매 자체는 소모되지 않는다 |
| 단백질 접힘 | "젖은 실이 뭉치듯 — 소수성 부분은 물을 피해 안으로" | 소수성 상호작용, 수소결합, 이온결합이 협력하여 3D 구조를 결정. 10^300개 가능한 구조 중 단 하나 (레빈탈 역설) |
| DNA 이중나선 | "나선형 사다리 — 가로대(염기쌍)의 순서가 유전 정보" | A-T(2개 H결합), G-C(3개 H결합) 상보적 결합. 나선 구조가 정보를 보호하고 복제를 가능하게 한다 |
| 르샤틀리에 | "기울어진 시소가 다시 균형을 찾으려 하는 것" | 평형 상태에 외부 변화(압력/온도/농도)를 주면 변화를 상쇄하는 방향으로 이동 |
| 결정 구조 | "레고 블록 — 같은 블록도 쌓는 방식에 따라 다른 구조물" | 동일 원소도 결정 배열에 따라 전혀 다른 물성 (흑연 vs 다이아몬드) |
| 배향 효과 | "열쇠와 자물쇠 — 맞는 방향으로 넣어야 열린다" | 분자 충돌 시 올바른 3D 방향으로 부딪쳐야 반응이 일어난다 |
