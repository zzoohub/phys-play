# PhysPlay Curriculum

4개 과학 트랙, 24개 모듈로 구성된 3D/XR 체험형 커리큘럼.

---

## 트랙 구성

| # | 트랙 | 스케일 | 핵심 경험 | 수학 의존도 | 타겟 |
|---|------|--------|-----------|------------|------|
| 1 | [고전물리](./track-1-classical/) | cm ~ km | 실험·시뮬레이션 | ⭐⭐⭐ 높음 | 중고생 |
| 2 | [화학](./track-2-chemistry/) | nm ~ μm | 조립·분해 | ⭐⭐ 보통 | 중고생, 대학 교양 |
| 3 | [우주과학](./track-3-space/) | AU ~ Gly | 줌인·줌아웃 | ⭐ 낮음 | 전연령 |
| 4 | [양자역학](./track-4-quantum/) | 양자 스케일 | 관측·확률 | ⭐⭐ 보통 | 고교생~성인, 테크 관심층 |

### 추천 학습 순서 (수학에 약한 학생)

> **우주과학 → 화학 → 양자역학 → 고전물리**
>
> 흥미 점화 → 개념 확장 → 직관 도전 → 수학 기반 심화

## 3D vs XR 전략

| 구분 | 3D (기본) | XR (보너스) |
|------|-----------|-------------|
| 접근성 | 브라우저만 있으면 OK | HMD 필요 |
| 사용자 풀 | 100% | 5~10% |
| 체험 품질 | 충분히 좋음 | 몰입감 극대화 |
| 설계 방향 | 마우스/터치 인터랙션 | 핸드 트래킹, 공간 이동 |

> 커리큘럼의 90%는 3D로 완전히 동작하며,
> XR은 "스케일 체감"과 "공간 몰입"이 중요한 콘텐츠에서 차이를 만든다.

## 콘텐츠별 XR 필요도

| 표시 | 의미 |
|------|------|
| 🟢 | 3D만으로 충분 |
| 🟡 | XR이면 더 좋지만 3D로도 OK |
| 🔴 | XR일 때 체험이 크게 달라짐 |

| 콘텐츠 | 필요도 | 비고 |
|--------|--------|------|
| 분자 회전·조작 | 🟢 | 마우스 드래그로 충분 |
| 시뮬레이션 (충돌, 파동, 회로) | 🟢 | 슬라이더 인터랙션 |
| 벡터장 시각화 | 🟢 | 3D 뷰어로 OK |
| 이중슬릿, 블로흐 구 | 🟢 | 인터랙티브 3D로 충분 |
| 타임랩스 (별의 진화 등) | 🟢 | 화면에서도 잘 동작 |
| 세포/DNA 내부 진입 | 🟡 | 몰입감 차이 |
| 태양계/은하 탐험 | 🟡 | 공간감 차이 |
| 우주 스케일 비교 | 🔴 | 모니터로는 스케일 체감 어려움 |
| 목성 옆에 서기 | 🔴 | VR일 때 압도적 경험 |
